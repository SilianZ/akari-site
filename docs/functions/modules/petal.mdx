# petal

此模块可以获取当前群组的花瓣数，花瓣可用于 [`ask`](ask) 模块。

## `petal`

- [公开](permissions)

用法：`~petal [<群组 ID>]`

此命令可以获取所在群组的花瓣数量，若拥有超级用户权限则可查看其他群组的花瓣数量。

## `petal modify`

- [仅超级用户](permissions)

用法：`~petal modify <数量> [<群组 ID>]`

此命令可以修改目标群组的花瓣数量，若 ID 留空则修改所在群组的花瓣数量。
