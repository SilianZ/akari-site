# hitokoto



此模块可以获取一言。

信息源来自[一言](https://hitokoto.cn/)。

## `hitokoto`

-   [公开](permissions)
-   别名：`htkt`

用法：`~hitokoto [<类型>]`

此命令可以根据类型获取一言，类型若留空则随机。

| 参数 | 说明     |
| ---- | -------- |
| `a`  | 动画     |
| `b`  | 漫画     |
| `c`  | 游戏     |
| `d`  | 文学     |
| `e`  | 原创     |
| `f`  | 来自网络 |
| `g`  | 其他     |
| `h`  | 影视     |
| `i`  | 诗词     |
| `j`  | 网易云   |
| `k`  | 哲学     |
| `l`  | 抖机灵   |

:::info 例子

返回的结果示例：

```
>>> ~hitokoto
<<< 花早晚会凋谢，记忆最终也会消散。
——「镇魂街」
类型：动画
https://hitokoto.cn?id=6106
```

:::
