# chunithm

-   [闭源](help/faq#为什么有些模块没有开源)
-   别名：`chu`

此模块可以查询 CHUNITHM 游戏中的信息。

如非特别说明，本页面的难度支持全称、三字母缩写和中文缩写（绿黄红紫黑），字母大小写不敏感。

注意

本模块的命令使用的数据源为 Diving-Fish 提供的[查分器](https://www.diving-fish.com/chunithmdx/prober/)，第一次使用本命令前请先按照对应说明注册账号并设置。

## `chunithm base`

-   [公开](permissions)

用法：`~chunithm base <定数> [<最大定数>]`

此命令可以根据定数搜索对应歌曲。

## `chunithm level`

-   [公开](permissions)

用法：`~chunithm level <等级> [<页数>]`

此命令可以根据等级搜索对应歌曲。

页数留空则默认为第 1 页。

## `chunithm search`

-   [公开](permissions)

用法：`~chunithm search <搜索词>`

此命令可以根据歌曲名称（或一部分）搜索对应歌曲。

## `chunithm id`

-   [公开](permissions)

用法：`~chunithm id <ID> [<难度>]`

此命令可以根据 ID 查询歌曲或谱面信息。

:::info 例子

返回的结果示例：

歌曲：

```
>>> ~chunithm id 1086
<<< 1086. 祈 -我ら神祖と共に歩む者なり-
艺术家：光吉猛修 VS 穴山大輔 VS Kai VS 水野健治 VS 大国奏音
分类：ORIGINAL
BPM：219
版本：CHUNITHM PARADISE LOST
难度：6.0/10.0/14.5/15.4
```

谱面：

```
>>> ~chunithm id 1086 Master
<<< 1086. 祈 -我ら神祖と共に歩む者なり-
Master 15 (15.4)
连击：4444
谱师：譜面ボーイズからの祈り -Pray for "JUSTICE"-
```

:::

## `chunithm song`

-   [公开](permissions)

用法：`~chunithm song <ID 或标题> [<难度>]`

此命令可以根据 ID 或别名查询歌曲或谱面信息。

ID 需要加入前缀“id”，标题请使用下划线“_”代替空格。

:::info 例子

返回的结果示例：

歌曲：

```
>>> ~chunithm song id1086
<<< 1086. 祈 -我ら神祖と共に歩む者なり-
艺术家：光吉猛修 VS 穴山大輔 VS Kai VS 水野健治 VS 大国奏音
分类：ORIGINAL
BPM：219
版本：CHUNITHM PARADISE LOST
难度：6.0/10.0/14.5/15.4
```

谱面：

```
>>> ~chunithm song id1086 Master
<<< 1086. 祈 -我ら神祖と共に歩む者なり-
Master 15 (15.4)
连击：4444
谱师：譜面ボーイズからの祈り -Pray for "JUSTICE"-
```

:::

## `chunithm random`

-   [公开](permissions)

用法：`~chunithm random [<等级+难度>]`

此命令可以随机一首指定条件的歌曲，若不输入参数则完全随机。难度和等级之间不需要空格。

## `chunithm b30`

-   [公开](permissions)

用法：`~chunithm b30 [<查分器用户名>]`

此命令可以用于获取 B30 信息。

在 QQ 平台，用户名留空则默认为绑定 QQ 号对应的用户名。

## `chunithm bind`

-   [公开](permissions)
-   Discord
-   Telegram

用法：`~maimai bind <用户名>`

此命令可以用于绑定 Diving-Fish 查分器用户。

## `chunithm unbind`

-   [公开](permissions)
-   Discord
-   Telegram

用法：`~maimai unbind`

此命令可以用于解除绑定 Diving-Fish 查分器用户。
