# random

-   [开源](/docs/help/faq#为什么有些模块没有开源)
-   别名：`rand`、`rng`

此模块可以从区间内生成随机数，或从集合中随机选择一个元素。

:::warning

此命令易被滥用，请勿在短时间内使用太多次，否则会触发[滥用警告](abuse_warning)。滥用命令是违反[服务条款](policy)的行为，不仅会影响正常交流，还有可能导致机器人账号被风控。

:::

## `random number`

-   [公开](permissions)

用法：`~random number <最小值> <最大值>`

此命令可以从区间内随机生成一个整数。

## `random choice`

-   [公开](permissions)

用法：`~random choice <元素 1> <元素 2> [<元素 3> <元素 4>...]`

此命令可以从集合内随机选择一个元素。

## `random shuffle`

-   [公开](permissions)

用法：`~random shuffle <元素 1> <元素 2> [<元素 3> <元素 4>...]`

此命令可以将集合中的所有元素重新排列。

## `random uuid`

-   [公开](permissions)

用法：`~random uuid`

此命令可以生成一个随机 UUID。
