# ab

-   [开源](help/faq#为什么有些模块没有开源)

此模块可以查询设定的起始 Wiki 的滥用日志。

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::

## `ab`

-   [公开](/docs/permissions)
-   转发消息

此命令可以查询滥用日志。

用法：`~ab`

## `ab legacy`

-   [公开](/docs/permissions)

用法：`~ab legacy`

此命令可以强制使用文字模式查询滥用日志。
