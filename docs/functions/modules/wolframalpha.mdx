# wolframalpha

-   [开源](help/faq#为什么有些模块没有开源)
-   别名：`wolfram`、`wa`

此模块可以使用 [WolframAlpha](https://wolframalpha.com/) 搜索问题，仅支持英文。

## `wolframalpha`

-   [公开](permissions)

用法：`~wolframalpha <问题或公式>`

此命令可以使用 WolframAlpha 搜索公式或问题，并以图片形式输出。

:::info 例子

返回的结果示例：

```
>>> ~wolframalpha lim(x->0) x/sin(x)
```

![](https://z1.ax1x.com/2023/11/12/piG8jqe.png)

:::

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::

## `wolframalpha ask`

-   [公开](permissions)

用法：`~wolframalpha ask <问题>`

此命令可使用 WolframAlpha 回答问题。

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::
