# mkey

-   [闭源](help/faq#为什么有些模块没有开源)

此模块可以计算任天堂系列主机监护管理的万能密码。

## `mkey`

-   [公开](permissions)

用法：`~mkey <主机> <月份> <日期> <确认代码> [<主机 ID>]`

此命令可以计算任天堂监护管理的万能密码。

支持的主机：`3ds`、`dsi`、`wii`、`wiiu`、`switch`。

![](https://z1.ax1x.com/2023/11/18/piNEJwq.png)

月份与日期为主机内已设置的日期，确认代码可在此处找到（以 Switch 为例）。

Switch 主机需要额外的主机 ID。
