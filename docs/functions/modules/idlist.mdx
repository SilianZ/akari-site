# idlist



此模块可以查询基岩版 ID。

信息源来自[命令助手](https://ca.projectxero.top/)。

## `idlist`

-   [公开](permissions)

用法：`~idlist <描述> [<过滤器>...]`

此命令可以查询 MCBEID 表。

| 中文           | 英文                |
| -------------- | ------------------- |
| 方块           | block               |
| 物品           | item                |
| 实体           | entity              |
| 状态效果       | effect              |
| 附魔类型       | enchant             |
| 迷雾           | fog                 |
| 结构           | location            |
| 实体事件       | entityevent         |
| 实体族         | entityfamily        |
| 动画           | animation           |
| 动画控制器     | animationcontroller |
| 粒子发射器     | particleemitter     |
| 声音           | sound               |
| 游戏规则       | gamerule            |
| 槽位类型       | entityslot          |
| 战利品表       | loottable           |
| 音乐           | music               |
| 可再生的实体   | summonableentity    |
| 战利品使用工具 | loottool            |
