# twenty_four


-   别名：`twentyfour`、`24`

此模块可以游玩 24 点。

:::warning

此命令易被滥用，请勿在短时间内使用太多次，否则会触发[滥用警告](abuse_warning)。滥用命令是违反[服务条款](policy)的行为，不仅会影响正常交流，还有可能导致机器人账号被风控。

:::

## `twenty_four`

-   [公开](permissions)

用法：`~twenty_four`

此命令可以生成 4 个数字，要求发起者使用所有数字组成一个结果为 24 的数学表达式。若数字组合无解，可发送“无解”。

若回答正确可获得 2 片花瓣，“无解”回答错误则失去 1 片花瓣。

支持的运算符有`+`、`-`、`*`、`/`和`()`，关于数学表达式的帮助见[数学表达式](help/expressions)。

:::info 例子

返回的结果示例：

```
>>> ~twenty_four
<<< 给出的数字组合：[1, 3, 5, 7]
请输入表达式使其结果为 24。（若无解，请输入“无解”）
```

正确回答示例：

```
>>> (1+5)*(7-3)
<<< 回答正确。
```

错误回答示例：

```
<<< 回答错误。
```

:::

## `twenty_four stop`

-   [公开](permissions)

用法：`~twenty_four stop`

此命令在模块正在运行时使用，则会停止。
