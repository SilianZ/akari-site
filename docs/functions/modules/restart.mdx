# restart

-   强制开启
-   [仅超级用户](permissions)

此模块可以更新或重启机器人。

## `restart`

-   [仅超级用户](permissions)

用法：`~restart`

此命令可以重启机器人。

## `update`

-   [仅超级用户](permissions)

用法：`~update`

此命令可以从 GitHub 拉取机器人新版本并更新。

## `update&restart`

-   [仅超级用户](permissions)
-   别名：`u&r`

用法：`~update&restart`

此命令可以从 GitHub 拉取机器人新版本并更新，之后重启机器人。
