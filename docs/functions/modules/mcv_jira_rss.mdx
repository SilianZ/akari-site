# mcv_jira_rss

-   [开源](help/faq#为什么有些模块没有开源)

此模块可以在 Mojira（Minecraft 的工作室——Mojang Studios 的漏洞汇报器）有新的 Minecraft 版本更新时发送通知。

## mcv_jira_rss

-   [订阅](/docs/help/reading#命令类型)

此模块可以在 Mojira 有新的 Minecraft：Java 版更新时发送通知。

::info 例子

这是机器人在版本更新时发送的消息示例：

```
Mojira 已更新 21w05b。
（Mojira 上的信息仅作版本号预览用，不代表启动器已更新此版本）
```

:::
