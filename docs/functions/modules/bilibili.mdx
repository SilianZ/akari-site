# bilibili


-   别名：`bili`

此模块可以查询关于哔哩哔哩视频网站上的视频相关信息。

## `bilibili`

-   [公开](permissions)

用法：

-   `~bilibili <视频编号> [-i]`
-   直接发送：`<视频编号>`
-   URL：`http[s]://b23.tv/<视频编号>`

参数：

-   `-i`：显示详细内容

此命令可以查看 AV 号或 BV 号对应的视频内容。

:::info 例子

返回的结果示例：

```
>>> ~bilibili BV1j4411W7F7
<<< https://www.bilibili.com/video/BV1j4411W7F7
标题：【猛男版】新 宝 岛
类型：宅舞
UP 主：果厨果厨果
日期：2019-05-28 13:19:24
```

详细内容：

```
>>> ~bilibili BV1j4411W7F7 -i
<<<   https://www.bilibili.com/video/BV1j4411W7F7

UP主：果厨果厨果 | 粉丝：246.3k

喜欢：5.3M | 投币：2.2M | 收藏：3.0M | 分享：2.2M
日期：2019-05-28 13:19:24
```

:::
