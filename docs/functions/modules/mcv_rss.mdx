# mcv_rss

-   [开源](/docs/help/faq#为什么有些模块没有开源)

此模块可以在 Minecraft 更新时发送通知。

## mcv_rss

-   [订阅](/docs/help/reading#命令类型)

此模块可以在 Minecraft：Java 版更新时发送通知。

:::info 例子

这是机器人在版本更新时发送的消息示例：

```
启动器已更新 21w05b 快照。
```

:::

## mcbv_rss

-   [订阅](/docs/help/reading#命令类型)

此模块可以在 Minecraft 基岩版更新时发送通知。

:::info 例子

这是机器人在版本更新时发送的消息示例：

```
基岩版商店已更新 1.10.19.23 正式版。
```

:::
