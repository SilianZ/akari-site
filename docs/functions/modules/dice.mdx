# dice

此模块可用于模拟掷骰子。

:::warning

此命令易被滥用，请勿在短时间内使用太多次，否则会触发[滥用警告](abuse_warning)。滥用命令是违反[服务条款](policy)的行为，不仅会影响正常交流，还有可能导致机器人账号被风控。

:::

## `dice`

-   [公开](permissions)
-   别名：`rd`、`roll`

用法：

-   `~dice <表达式> [<检定点数>]`
-   `(扔|投|掷|丢)<m>个[<n>面]骰子`

表达式参数：`[<N>x<m>]d<n>[(k|kl)<x>±<y>]`

-   `n`：所掷骰子的面数。
-   `m`：所掷骰子的个数（可选，默认为 1）
-   `N`：所掷骰子的次数（可选，默认为 1）
-   `kx`：优势骰，选取结果最大的 x 个骰子之和（可选）
-   `klx`：劣势骰，选取结果最小的 x 个骰子之和（可选）
-   `y`：调节值，将所得结果加减对应数值（可选）

此命令可以自定义掷骰子，并支持点数检定。

当检定点数存在时，若掷骰子结果符合检定点数则检定成功，反之检定失败。

骰子相关设置可在[配置文件](development/config)修改。

## `dice rule`

-   [仅管理员](permissions)
-   设置

用法：`~dice rule`

此命令可以切换检定点数的判断规则。
