# prefix

-   强制开启
-   [仅管理员](permissions)

此模块可以自定义命令前缀。

## `prefix add`

-   [仅管理员](permissions)

用法：`~prefix add <前缀>`

此命令可以设置自定义机器人命令前缀。

## `prefix remove`

-   [仅管理员](permissions)

用法：`~prefix remove <前缀>`

此命令可以移除自定义机器人命令前缀。

## `prefix reset`

-   [仅管理员](permissions)

用法：`~prefix reset`

此命令可以重置自定义机器人命令前缀。

## `prefix list`

-   [仅管理员](permissions)

用法：`~prefix list`

此命令可以列出自定义命令前缀列表。
