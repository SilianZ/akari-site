# summary


-   QQ 群

此模块可以给一段聊天记录生成摘要。

此模块由 InstructGPT 提供支持，使用了 GPT-3.5 中的 `text-davinci-003` 模型。

## `summary`

-   [公开](permissions)
-   QQ 群
-   冷却 60 秒

用法：`~summary`

此命令可以将一条 QQ 合并转发消息生成摘要。

发起命令后请发送一条合并转发消息，稍后将自动生成一句摘要。

:::tip 消息转发

在 QQ 中，若要发送合并转发消息，请长按需要转发的第一条消息，选择“多选”，点击需要转发的所有消息，然后选择第二个图标，点击“合并转发”，选择想要发送的群，点击发送即可。

:::

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::
