# tic_tac_toe


-   别名：`tictactoe`、`ttt`

此模块可以游玩井字棋。

发起时输入数字来代表选择格子位数，可发送单个 1-9 数字或两个 1-3 数字（分别代表行和列）。

:::warning

此命令易被滥用，请勿在短时间内使用太多次，否则会触发[滥用警告](abuse_warning)。滥用命令是违反[服务条款](policy)的行为，不仅会影响正常交流，还有可能导致机器人账号被风控。

:::

## `tic_tac_toe`

-   [公开](permissions)

用法：`~tic_tac_toe`

此命令可以与机器人玩井字棋，机器人可能会失误。

若胜利可以获得 1 片花瓣。

## `tic_tac_toe expert`

-   [公开](permissions)

用法：`~tic_tac_toe expert`

此命令可以与专家机器人玩井字棋，机器人偶尔会失误。

若胜利可以获得 2 片花瓣。

## `tic_tac_toe master`

-   [公开](permissions)

用法：`~tic_tac_toe master`

此命令可以与大师机器人玩井字棋，机器人不会失误，因此你不会胜利。

## `tic_tac_toe noob`

-   [公开](permissions)

用法：`~tic_tac_toe noob`

此命令可以与菜鸟机器人玩井字棋，机器人经常会失误。

## `tic_tac_toe duo`

-   [公开](permissions)

用法：`~tic_tac_toe duo`

此命令可以与双人玩井字棋。

## `tic_tac_toe stop`

-   [公开](permissions)

用法：`~tic_tac_toe stop`

此命令在模块正在运行时使用，则会停止。
