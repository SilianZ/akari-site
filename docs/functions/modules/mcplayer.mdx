# mcplayer

此模块可以使用 Mojang API 查询一个 Minecraft：Java 版玩家的信息。

## `mcplayer`

-   [公开](permissions)

用法：`~mcplayer <玩家名|UUID>`

此命令可以通过玩家名或 UUID 获取玩家信息。

:::info 例子

返回的结果示例：

```
>>> ~mcplayer jeb_
<<< jeb_（853c80ef3c3749fdaa49938b674adae6）
NameMC：https://namemc.com/profile/jeb_
```
