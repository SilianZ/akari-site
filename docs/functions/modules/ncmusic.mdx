# ncmusic

-   [开源](help/faq#为什么有些模块没有开源)

此模块可以用来搜索网易云音乐并查看歌曲详情。

## `ncmusic search`

-   [公开](permissions)

用法：`~ncmusic search <搜索词>`

此命令可以搜索网易云音乐的歌曲。

## `ncmusic search legacy`

-   [公开](permissions)

用法：`~ncmusic search legacy <搜索词>`

此命令可以强制使用文字模式搜索网易云音乐的歌曲。

## `ncmusic info`

-   [公开](permissions)

用法：`~ncmusic info <ID>`

此命令可以根据歌曲 ID 获取歌曲的信息。

:::info 例子

返回的结果示例：

```
>>> ~ncmusic info 5221167
<<< 歌名：Never Gonna Give You Up（5221167）
专辑名：Simply The Best Of The 80's（509806）
歌手：Rick Astley
歌曲详情页：https://music.163.com/#/song?id=5221167
```

:::
