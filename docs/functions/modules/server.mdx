# server


-   别名：`s`

此模块可以查询一个 Minecraft 服务器的信息。

## `server`

-   [公开](permissions)

用法：`~server <服务器 IP 或域名>[:<端口>] [-r] [-p]`

参数：

-   `-r`：显示原始信息
-   `-p`：显示玩家列表

此命令可以查询一个 Minecraft 服务器的信息。

:::info 例子

返回的结果示例：

常规：

```
>>> ~server hypixel.net
<<< [JE]
                Hypixel Network [1.8-1.19]
            3RD SKYBLOCK ANNIVERSARY
在线玩家：47649 / 200000
游戏版本：Requires MC 1.8 / 1.19
hypixel.net:25565
```

原始信息：

```
>>> ~server hypixel.net -r
<<< [JE]
                §aHypixel Network §c[1.8-1.19]
            §d§l3RD SKYBLOCK ANNIVERSARY
在线玩家：47666 / 200000
游戏版本：Requires MC 1.8 / 1.19
hypixel.net:25565
```

玩家列表：

```
>>> ~server hypixel.net -p
<<< [JE]
                Hypixel Network [1.8-1.19]
            3RD SKYBLOCK ANNIVERSARY
在线玩家：47636 / 200000
当前在线玩家：
（略）
游戏版本：Requires MC 1.8 / 1.19
hypixel.net:25565
```

:::
