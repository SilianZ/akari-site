# cave

-   [开源](/docs/help/faq#为什么有些模块没有开源)

此模块可以使用回声洞。

回声洞类似于留言板，用户可以储存自己的留言。

## `cave`

-   [公开](permissions)
-   冷却 90 秒

用法：

-   `~cave [<id>]`

此命令可以获取回声洞。

填入 id 可以指定回声洞编号。

:::info 例子

返回的结果示例：

```
>>> ~cave
<<< 回声洞 #33
蒙德
——Emerald-AM9
```

:::

## `cave add`

-   [公开](permissions)

用法：

-   `~cave add`

此命令可以添加回声洞。

## `cave list`

-   [仅超级用户](permissions)

用法：

-   `~cave list`

此命令可以获取回声洞列表。

:::info 例子

返回的结果示例：


## `cave remove`

用法：

-   `~cave remove <id>`
-   `~cave detele <id>`

此命令可以删除指定 id 的回声洞。
