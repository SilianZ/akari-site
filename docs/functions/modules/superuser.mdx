# superuser

此模块可以设置机器人超级用户。

## `superuser add`

- [仅超级用户](permissions)

用法：`~superuser add <用户 ID>`

此命令可以添加机器人超级用户。

## `superuser remove`

- [仅超级用户](permissions)

用法：`~superuser remove <用户 ID>`

此命令可以移除机器人超级用户。
