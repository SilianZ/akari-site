# echo

-   [开源](help/faq#为什么有些模块没有开源)
-   强制开启

此模块可以让机器人重复消息。

## `echo`

-   [仅超级用户](permissions)

用法：`~echo <消息>`

此命令可以重复消息。

## `say`

-   [仅超级用户](permissions)

用法：`~say <消息>`

此命令可以重复消息，但不进行回复。

:::info

机器人自己发送的消息并不会触发自己的其他命令。

:::

## `raise`

-   [仅超级用户](permissions)

此命令可以抛出异常进行测试。
