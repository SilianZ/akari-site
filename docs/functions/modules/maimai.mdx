# maimai

-   [闭源](help/faq#为什么有些模块没有开源)
-   别名：`mai`

此模块可以查询 Maimai 游戏中的信息。

本模块移植自 [Maibot](https://github.com/Diving-Fish/mai-bot) 。

如非特别说明，本页面的难度支持全称、三字母缩写和中文缩写（绿黄红紫白），字母大小写不敏感。

:::warning

本模块的命令使用的数据源为 Diving-Fish 提供的[查分器](https://www.diving-fish.com/maimaidx/prober/)，第一次使用本命令前请先按照对应说明注册账号并设置。

:::

## `maimai base`

-   [公开](permissions)

用法：`~maimai base <定数> [<最大定数>]`

此命令可以根据定数搜索对应歌曲。

## `maimai level`

-   [公开](permissions)

用法：`~maimai level <等级> [<页数>]`

此命令可以根据等级搜索对应歌曲。

页数留空则默认为第 1 页。

## `maimai new`

-   [公开](permissions)

用法：`~maimai new [<页数>]`

此命令可以获取当前版本最新歌曲。

页数留空则默认为第 1 页。

## `maimai search`

-   [公开](permissions)

用法：`~maimai search <搜索词>`

此命令可以根据歌曲名称（或一部分）搜索对应歌曲。

## `maimai alias`

-   [公开](permissions)

用法：`~maimai alias <ID>`

此命令可以查询歌曲的别名。歌曲别名库由 Xray 提供。

## `maimai id`

-   [公开](permissions)

用法：`~maimai id <ID> [<难度>]`

此命令可以根据 ID 查询歌曲或谱面信息。

:::info 例子

返回的结果示例：

歌曲：

```
>>> ~maimai id 834
<<< 834. PANDORA PARADOXXX
艺术家：削除
分类：舞萌
BPM：150
版本：maimai FiNALE
难度：7.7/11.5/13.9/14.9/15.0
```

谱面：

```
>>> ~maimai id 834 Re:MASTER
<<< 834. PANDORA PARADOXXX
Re:MASTER 15（15.0）
TAP：1165
HOLD：72
SLIDE：90
BREAK：15
谱师：PANDORA PARADOXXX
```

:::

谱面：

```

<<< 834. PANDORA PARADOXXX
Re:MASTER 15（15.0）
TAP：1165
HOLD：72
SLIDE：90
BREAK：15
谱师：PANDORA PARADOXXX

:::

## `maimai random`

-   [公开](permissions)

用法：

-   指定条件：`~maimai random <等级+难度> [<类型>]`
-   完全随机：`~maimai random`

此命令可以随机一首指定条件的歌曲，若不输入参数则完全随机。

如果不想指定难度和等级则输入 `*` 跳过，难度和等级之间不需要空格。

类型可输入 `SD` 或 `DX` 之一，留空或输入其他内容则不筛选。

## `maimai b50`

-   [公开](permissions)

用法：`~maimai b50 [beta] [<查分器用户名>]`

此命令可以查询 B50 信息。

在 QQ 平台，用户名留空则默认为绑定 QQ 号对应的用户名。

加入 `beta` 强制使用 Beta 版渲染。

## `maimai bind`

-   [公开](permissions)
-   Discord
-   Telegram

用法：`~maimai bind <用户名>`

此命令可以用于绑定 Diving-Fish 查分器用户。

## `maimai unbind`

-   [公开](permissions)
-   Discord
-   Telegram

用法：`~maimai unbind`

此命令可以用于解除绑定 Diving-Fish 查分器用户。

## `maimai info`

-   [公开](permissions)

用法：

- `~maimai info <ID 或别名> [<查分器用户名>]`

此命令可以根据 ID 或别名查询歌曲的谱面成绩。

ID 需要加入前缀“id”，别名请使用下划线“_”代替空格。

在 QQ 平台，用户名留空则默认为绑定 QQ 号对应的用户名。

:::info 例子

返回的结果示例：

```
>>> ~maimai info id834
<<< 834. PANDORA PARADOXXX
Basic 7+
100.7275 SSS+ FC
Advanced 11
100.7481 SSS+ FC
Expert 13+
100.5622 SSS+ FC
Master 14+
99.1487 SS
Re:MASTER 15
98.1166 S+
```

:::

:::tip 无法使用？

请检查你是否输入的是**查分器用户名**（非游戏名或昵称）。

:::

## `maimai plate`

-   [公开](permissions)

用法：`~maimai plate <名牌板> [<查分器用户名>]`

此命令可以查看玩家的名牌板完成进度。

在 QQ 平台，用户名留空则默认为绑定 QQ 号对应的用户名。

:::info 例子

Basic 剩余 524 首，
Advanced 剩余 524 首，
Expert 剩余 524 首，
Master 剩余 524 首，
Re:MASTER 剩余 84 首。
<<< 你还有 208 首等级大于 13+ 的曲目。
```

:::

:::tip 无法使用？

请检查你是否输入的是**查分器用户名**（非游戏名或昵称）。

:::

## `maimai process`

-   [公开](permissions)

用法：`~maimai process <等级> <目标> [<查分器用户名>]`

此命令可以查看玩家在对应等级的评价完成进度。

在 QQ 平台，用户名留空则默认为绑定 QQ 号对应的用户名。

支持的目标评价：

-   等级：`A`、`AA`、`AAA`、`S`、`S+`、`SS`、`SS+`、`SSS`、`SSS+`
-   连击：`FC`、`FC+`、`AP`、`AP+`
-   同步率：`FS`、`FS+`、`FDX`、`FDX+`

:::tip 无法使用？

请检查你是否输入的是**查分器用户名**（非游戏名或昵称）。

:::

## `maimai scoreline`

-   [公开](permissions)

用法：`~maimai scoreline <ID> <难度> <分数>`

此命令可以查找某首曲目的分数线，以及返回对应分数允许的 TAP GREAT 容错以及 BREAK 2500 或 BREAK 2000 相当的TAP GREAT 数。

:::info 例子

返回的结果示例：

```
>>> ~maimai scoreline 834 Re:MASTER 100
<<< PANDORA PARADOXXX Re:MASTER
分数线 100.0% 允许的最多 TAP GREAT 数量为 82.70（每个-0.0121%），
BREAK 2550（一共 15 个）相当于 1.378 个 TAP GREAT（-0.0167%），
BREAK 2000（一共 15 个）相当于 8.308 个 TAP GREAT（-0.1005%）。
```

:::
