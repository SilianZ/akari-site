# mcmod

-   [开源](/docs/help/faq#为什么有些模块没有开源)

此模块可以获取 Minecraft Mod 的信息。

## `mcmod`

-   [公开](permissions)

用法：`~mcmod <Mod 名称>`

此命令可以通过 Mod 名称获取 Mod 简介和链接，可使用无歧义缩写和准确中文。

## `mcmod details`

-   [公开](permissions)

用法：`~mcmod details <内容>`

此命令可以通过 Mod 内容的名称获取 Mod 简介和链接，可使用物品/方块/实体的 ID 或准确中文。
