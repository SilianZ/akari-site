# 阅读帮助

本页面可以帮助你阅读文档。

## 命令格式

以 `~wiki` 命令作为示例：

```
~wiki [<interwiki>:]<页面名>
```

`<页面名>` 代表**参数**。即使用 `<` 和 `>` 包含的参数，默认必填。

`[<interwiki>:]` 代表**选填参数**。即使用 `[` 和 `]` 包含的参数。通常情况下，帮助文档都会提及选填参数不填的情况下的默认值。

另外，带有 `(|)` 则代表**或**，即使用两种（及以上）选择的任意一种都可以响应命令。

错误示例

请勿模仿此命令，它不会被正确响应。

-   ~~`~wiki [:]<苦力怕>`~~

## 命令示例

命令示例有时会出现在页面中，以下是命令示例的格式，以 `~whoami` 命令作为示例：

:::info 例子

返回的结果示例：

```
>>> ~whoami
<<< 你的 ID 是：QQ|xxx
本对话的 ID 是：QQ|Group|xxx
```

其中，`>>>` 代表输入（即你发送给机器人的命令），`<<<` 代表命令的输出（即机器人发送给你的结果）。

:::

## 指导

指导通常会出现在文档和命令的开头。

### 用户权限

-   [公开](/docs/permissions)：此命令可公开使用。
-   [仅管理员](/docs/permissions)：此命令仅机器人管理员或群管理员可使用。
-   [仅超级用户](/docs/permissions)：此命令仅超级用户可使用。

### 命令类型

-   [命令](/docs/help/reading#命令类型)：此命令在使用时会立刻做出回应。（一般不显示）
-   [订阅](/docs/help/reading#命令类型)：此命令在使用后可开关消息推送。
-   [设置](/docs/help/reading#命令类型)：此命令在使用后可更改数据库中设定。

### 其它

-   强制开启：此命令为基础命令，无法被手动关闭和卸载。
-   （任意时间）后撤回：此命令输出的信息会在若干时间后撤回。
-   冷却（任意时间）：使用此命令后在若干时间后才可再次使用。
-   不推荐：不建议用户使用此命令。
