# 权限

小可内置了以下权限组：

-   `superuser`：超级用户，即全域机器人管理员，仅被开发者使用。
-   `admin`：机器人管理员，群内管理员或私聊默认拥有此权限。
-   `user`：普通用户。

在本文档中的权限标识对应关系如下：

| 标识       | 对应的权限                |
| -------- | -------------------- |
| 仅管理员     | `superuser`和`admin`  |
| 仅超级用户    | `superuser`          |
| 公开       | `user`               |

## 另见

如果你想设置权限，请使用以下命令：
