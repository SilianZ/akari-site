# color

-   别名：`colour`

此模块可以提供颜色信息。

## `color`

-   [公开](permissions)

用法：`~color [<颜色>]`

此命令可以给出对应颜色的颜色信息。

本命令支持以下颜色格式：

-   16 进制（`#`可省略）：`#123456`、`#123`
-   RGB：`rgb(12,34,56)`
-   HSL：`hsl(123,45%,67%)`
-   CSS 颜色（[列表](color_list#CSS)）：`blue`
-   Material Design 颜色（[列表](color_list#material-design)）：`blue-500`

留空则随机颜色。

若颜色存在于 CSS 颜色或 Material Design 颜色中，则图片内会显示其对应的颜色代号。

:::info 例子

返回的结果示例：

![](https://z1.ax1x.com/2023/11/06/pilBEkQ.png)

```
>>> ~color #0091ff
<<< HEX：#0091ff
RGB：rgb(0, 145, 255)
HSL：hsl(205, 100%, 50%)
最相似的 CSS 颜色名称: dodgerblue
最相似的 Material Design 颜色名称: lightblue-A700
```

:::
