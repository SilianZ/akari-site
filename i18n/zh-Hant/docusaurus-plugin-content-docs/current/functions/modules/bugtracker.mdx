# bugtracker

此模块可以查询一个来自 Mojira（Minecraft 的开发工作室 Mojang Studios 的漏洞汇报器）的指定漏洞汇报的详情。

## `bugtracker`

- [公开](permissions)
- 别名：`bug`

用法：

- `~bugtracker <漏洞 ID>`
- 直接漏洞 ID：`<漏洞 ID>`

此命令可以查询一个来自 Mojira 的指定漏洞汇报的详情。

本命令无法查询隐私漏洞。

:::info 例子

返回的结果示例：

```
>>> ~bugtracker MC-4
<<< [MC-4] Item drops sometimes appear at the wrong location
Type: Bug
Project: Minecraft: Java Edition
Status: Reopened
Mojang Priority: Low
Resolution: Unresolved
Versions: Minecraft 1.4.1 ~ 1.20 Pre-release 6
https://bugs.mojang.com/browse/MC-4
```

:::
