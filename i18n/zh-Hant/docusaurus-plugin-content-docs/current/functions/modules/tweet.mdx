# tweet

-   [开源](/docs/help/faq#为什么有些模块没有开源)

此模块可以生成一个推文摘要。

## `tweet`

-   [公开](permissions)
-   别名：`x`

用法：`~tweet <推文 ID>`

此命令可以获取并生成一个推文的图片摘要，并支持显示喜欢数据。

:::info 例子

返回的结果示例：

```
>>> ~tweet 1625695877326340102
<<<  https://twitter.com/elonmusk/status/1625695877326340102
```

![](https://z1.ax1x.com/2023/11/18/piN0JFx.png)

:::

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::
