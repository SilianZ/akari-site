# superuser

-   [开源](/docs/help/faq#为什么有些模块没有开源)
-   强制开启

此模块可以设置机器人超级用户。

## `superuser add`

-   [仅超级用户](permissions)

用法：`~superuser add <用户 ID>`

此命令可以添加机器人超级用户。

## `superuser remove`

-   [仅超级用户](permissions)

用法：`~superuser remove <用户 ID>`

此命令可以移除机器人超级用户。
