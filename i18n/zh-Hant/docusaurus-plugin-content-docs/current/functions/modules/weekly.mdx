# weekly

-   [开源](help/faq#为什么有些模块没有开源)

此模块可以获取中文 Minecraft Wiki 的每周页面或茶馆周报。

## `weekly`

-   [公开](permissions)

用法：`~weekly`

此模块可以获取中文 Minecraft Wiki 的每周页面。

## `weekly teahouse`

-   [公开](permissions)

用法：`~weekly teahouse`

此模块可以获取茶馆周报。
