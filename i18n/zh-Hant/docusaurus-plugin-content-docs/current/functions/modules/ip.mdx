# ip

-   [开源](/docs/help/faq#为什么有些模块没有开源)

此模块可以查询 IP 的信息。

## `ip`

-   [公开](permissions)

用法：`~ip <IP>`

此命令可以查询 IP 的信息。

:::info 例子

返回的结果示例：

```
>>> ~ip 8.8.8.8
<<< 8.8.8.8
类型：IPv4 全局地址
位置：Los Angeles, California, United States (-118.2441°W, 34.0544°N)
邮编：90009
组织：Google
ASN：15169 (GOOGLE)
时区：UTC-8
反向解析：dns.google
```

:::
