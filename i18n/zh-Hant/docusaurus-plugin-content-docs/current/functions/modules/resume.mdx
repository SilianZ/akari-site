# resume

-   [开源](help/faq#为什么有些模块没有开源)
-   强制开启

此模块用于重发消息。

## `resume`

-   [仅超级用户](permissions)

用法：`~resume`

此命令可以重发消息。

有时候，由于 QQ 的风控问题，导致机器人无法正常发送消息，使用此命令可以重发这些消息。

## `resume clear`

-   [仅超级用户](permissions)

用法：`~resume clear`

此命令可以清除所有需要重发的消息。

## `resume continue`

-   [仅超级用户](permissions)

用法：`~resume continue`

此命令可以重发所有需要重发的消息。
