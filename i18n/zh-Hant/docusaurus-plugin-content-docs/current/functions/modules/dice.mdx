# dice

此模块可用于投掷骰子。

## `dice`

-   [公开](permissions)

-   别名：`rd`

用法：

-   `dice <多项式> [<检定点数>]`
-   `(扔|投|掷|丢)<m>个[<n>面]骰子`

此命令可以自定义投掷骰子，并支持点数检定。

当检定点数存在时，若掷骰子结果大于检定点数则显示成功，反之显示失败。

相关设置可在[配置文件](dev/config)修改。

骰子多项式参数：`[<N>#<m>]d<n>[(k|kl)<x>±<y>]`

-   `n`：所掷骰子的面数。
-   `m`：所掷骰子的个数（可选，默认为 1）
-   `N`：所掷骰子的次数（可选，默认为 1）
-   `kx`：优势骰，选取结果最大的 x 个骰子之和（可选）
-   `klx`：劣势骰，选取结果最小的 x 个骰子之和（可选）
-   `y`：调节值，将所得结果加减对应数值（可选）

:::caution

此命令易被滥用，请勿在短时间内使用太多次，否则会触发[滥用警告](abuse_warning)。滥用命令是违反[服务条款](https://bot.teahouse.team/wiki/服务条款)的行为，不仅会影响正常交流，还有可能导致机器人账号被风控。

:::
