# whois

-   [开源](help/faq#为什么有些模块没有开源)

此模块可以查询域名的 WHOIS。

## `whois`

-   [公开](permissions)

用法：`~whois <域名>`

此命令可以查询 WHOIS。

:::info 例子

返回的结果示例：

```
>>> ~whois google.com
<<< 域名名称：google.com
注册商：MarkMonitor, Inc.
WHOIS 服务器：whois.markmonitor.com
更新日期：2019-09-09 15:39:04
注册日期：1997-09-15 04:00:00
到期日期：2028-09-14 04:00:00
DNS 服务器：ns1.google.com, ns3.google.com, ns2.google.com, ns4.google.com
DNSSEC：unsigned
组织：Google LLC
位置：CA, US
```

:::
