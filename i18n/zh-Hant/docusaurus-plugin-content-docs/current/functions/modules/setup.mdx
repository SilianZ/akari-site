# setup

- 强制开启
- 别名：`toggle`

此模块可以设置机器人在群组内的行为。

## `setup typing`

- 设置

用法：`~setup typing`

此命令可以切换是否显示输入提示。

在 QQ 显示为戳一戳，在 Discord 上显示为“正在输入...”

## `setup timeoffset`

- [仅管理员](permissions)
- 设置

用法：`~setup timeoffset <偏移值>`

此命令可以设置当前群组的时间偏移值。
