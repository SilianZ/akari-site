# mcv

此模块可以查询 Minecraft 的最新版本。

## `mcv`

- [公开](permissions)
- 别名：`m`

用法：`~mcv`

此命令可以查询 Minecraft：Java 版启动器记载的最新正式、快照版本和 Mojira（Mojang Studios 的漏洞汇报器）记载的最新正式、快照版本。

## `mcbv`

- [公开](permissions)

用法：`~mcbv`

此命令可以查询 Minecraft 基岩版在 Mojira（Mojang Studios 的漏洞汇报器）记载的最新版本。

## `mcdv`

- [公开](permissions)

用法：`~mcdv`

此命令可以查询 Minecraft Dungeons 在 Mojira（Mojang Studios 的漏洞汇报器）记载的最新版本。

## `mcev`

- [公开](permissions)

用法：`~mcev`

此命令可以查询 Minecraft：教育版的 Windows 版记载的最新版本。

## `mclgv`

- [公开](permissions)

用法：`~mclgv`

此命令可以查询 Minecraft Legends 在 Mojira（Mojang Studios 的漏洞汇报器）记载的最新版本。
