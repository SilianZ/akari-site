# set

- [仅超级用户](permissions)

此模块可以操作机器人数据库。

## `set modules`

- [仅超级用户](permissions)

用法：`~set modules <目标 ID> <模块>`

此命令可以为对象配置模块。

## `set module disable`

- [仅超级用户](permissions)

用法：`~set module disable <目标 ID> <模块 1> [<模块 2>...]`

此命令可以为对象设置参数。

## `set option`

- [仅超级用户](permissions)

用法：`~post_whitelist <群组 ID>`

此命令可以查看目标的参数，键名为空则查看所有参数。

## `post_whitelist`

- [仅超级用户](permissions)

用法：`~set option <目标 ID> <键名> <值>`

此命令可以修改目标的参数。

## 强制开启

- [仅超级用户](permissions)

此命令可以设置 QQ 群组的推送白名单。

此命令可以移除目标的参数。
