# set

- [仅超级用户](permissions)

此模块可以操作机器人的数据库。

## `set module enable`

- [仅超级用户](permissions)

用法：`~set module enable <目标 ID> <模块 1> [<模块 2>...]`

此命令可以为目标打开模块。

## `set module disable`

- [仅超级用户](permissions)

用法：`~set module disable <目标 ID> <模块 1> [<模块 2>...]`

此命令可以为目标关闭模块。

## `set option get`

- [仅超级用户](permissions)

用法：`~set option get <目标 ID> [<键名>]`

此命令可以查看目标的参数，键名为空则查看所有参数。

## `set option edit`

- [仅超级用户](permissions)

用法：`~set option edit <目标 ID> <键名> <值>`

此命令可以修改目标的参数。

## `set option remove`

- [仅超级用户](permissions)

用法：`~set option remove <目标 ID> <键名>`

此命令可以移除目标的参数。
