# arcaea

此模块可以查询 Arcaea 游戏中的信息。

:::info

根据 Arcaea TOS 3.2.6 规定，歌曲查分功能将无限期停用，详见[此处](https://arcaea.lowiro.com/zh/terms_of_service)。

:::

## `arcaea random`

-   [公开](https://bot.teahouse.team/wiki/权限)

用法：`~arcaea random`

此模块可以让随机一首 Arcaea 曲目。

## `arcaea rank free`

-   [公开](https://bot.teahouse.team/wiki/权限)

用法：`~arcaea rank free`
