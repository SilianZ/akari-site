# coin

-   [开源](/docs/help/faq#为什么有些模块没有开源)

此模块可用于投掷硬币或打水漂。

:::warning

此命令易被滥用，请勿在短时间内使用太多次，否则会触发[滥用警告](abuse_warning)。滥用命令是违反[服务条款](policy)的行为，不仅会影响正常交流，还有可能导致机器人账号被风控。

:::

## `coin`

-   [公开](permissions)

用法：

-   `~coin <硬币数量>`
-   `(丢|抛)[<硬币数量>(个|枚)]硬币`

此命令可以投掷任意数量的硬币。

默认硬币数量最大为 10000 枚，正反面概率为 199/400（49.75%），直立概率为 1/200（0.5%）。

硬币最大数量和正反面概率可在[配置文件](development/config)修改。

## `stone`

-   [公开](permissions)
-   冷却 30 秒

用法：`~stone`

此命令可以模拟打水漂。

打水漂的数量在 0-10 浮动，若结果为 10 可以得到 1 片花瓣。
