# superuser

-   [开源]\(help/faq#Why some modules are not open sources)
-   Force turn on

This module can set robot superusers.

## `superuser add`

-   [仅超级用户](permissions)

Use：`~superuser add <User ID>`

This command can add robot superusers.

## `superuser remove`

-   [仅超级用户](permissions)

Use：`~superuser remove <User ID>`

This command can remove robot superusers.
