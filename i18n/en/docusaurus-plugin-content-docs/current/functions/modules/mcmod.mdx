# mcmod

-   [开源]\(/docs/help/faq#Why some modules are not open sources)

This module can get Minecraft Mod information.

## `mcmod`

-   [公开](permissions)

Use：`~mcmod <Mod 名称>`

This command can fetch mod profiles and links through mod names. It can be used in unambiguous acronyms and accurate Chinese.

## `mcmod details`

-   [公开](permissions)

Use：`~mcmod details <content>`

This command can fetch mod profiles and links through the name of mod content. It can use the IDs of items/blocks/entities or accurate Chinese.
