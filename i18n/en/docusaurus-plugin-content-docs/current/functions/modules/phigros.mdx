# phigros

-   [闭源]\(/docs/help/faq#Why some modules are not open sources)
-   Alias：`p`, `pgr`, `phi`

This module can query information in the Phigros game.

## `phigros b19`

-   [公开](permissions)

Use：`~phigros b19`

This command can fetch Best19 information from a Phigros user.

:::info Example

Result Example：

```
>>> ~phigros b19
```

:::

## `phigros bind`

-   [公开](permissions)
-   Revoke in 15 seconds

Use：`~phigros bind <sessionToken>`

This command can be used to bind to Phigros users.

About session Token

`sessionToken` is a string of lowercase letters and numbers with a length of 25 and can be used to fetch cloud archive data.

Since `sessionToken` is a private message, please bind as much as possible through private chat, otherwise the spoiler may destroy or tamper with your cloud archival data.If `sessionToken` is leaked, exit and re-login to Phigros should be reset as soon as possible.

Access to `sessionToken` and other information can be found in[此处](https://mivik.gitee.io/pgr-bot-help/index.html#Android).

## `phigros unbind`

-   [公开](permissions)

Use：`~phigros unbind`

This command can be used to unbind Phigros users.

## `phigros update`

-   [仅超级用户](permissions)

Use：`~phigros update`

This command can update Phigros resource files, such as sketch and quantity.
