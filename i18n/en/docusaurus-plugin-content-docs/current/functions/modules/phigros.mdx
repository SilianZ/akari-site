# phigros

- [闭源](help/faq#为什么有些模块没有开源)
- 别名：`p`、`pgr`、`phi`

此模块可以查询 Phigros 游戏中的信息。

## `phigros b19`

- [公开](permissions)

用法：`~phigros b19`

此命令可以获取一个 Phigros 用户的 Best19 信息。

:::info 例子

返回的结果示例：

```
>>> ~phigros b19
```

![](https://z1.ax1x.com/2023/11/18/piNuwp6.png)

:::

## `phigros bind`

- [公开](permissions)
- 15 秒后撤回

用法：`~phigros bind <sessionToken>`

此命令可以用于绑定 Phigros 用户。

关于 sessionToken

`sessionToken` 是一个长度为 25 的由小写英文字母和数字构成的字符串，可用于获取对应的云存档数据。

由于 `sessionToken` 为隐私信息，请尽可能通过私聊的方式完成绑定，否则破坏者可能会破坏或篡改你的云存档数据。若 `sessionToken` 已泄露，应尽快退出并重新登录 Phigros 以重置。

关于 `sessionToken` 的获取以及其他信息，可前往[此处](https://mivik.gitee.io/pgr-bot-help/index.html#安卓)。

## `phigros unbind`

- [公开](permissions)

用法：`~phigros unbind`

此命令可以用于解除绑定 Phigros 用户。

## `phigros update`

- [仅超级用户](permissions)

用法：`~phigros update`

此命令可以更新 Phigros 资源文件，如曲绘和定数。
