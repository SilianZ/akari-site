# bilibili

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Alias：`bili`

This module will be able to find information about videos on the beep mile beep.

## `bilibili`

-   [公开](permissions)

Usage：

-   \`~bilibili <video number> [-i]
-   Enter：`<video number>` directly
-   Use URL：
    -   完整链接：`http[s]://www.bilibili.com/video/<视频编号>`
    -   Short link：`HTML[s]://b23.tv/<video number>`

Parameter：

-   `-i`：shows details

This command can view the video content of the AV or BV number.

:::info Example

Result Example：

```
>>> ~bilibili BV1j4411W7F7
<<https://www.bilibili>. om/video/BV1j4411W7F7
Heading：'Lang' New
Type：Home Dance
UP Owner：fruit kitchen
date：2019-05-28 13:19:24
```
