# config

-   [开源]\(help/faq#Why some modules are not open sources)
-   Alias：`cfg`

This module can operate robot[配置文件](development/config).

## `config write`

-   [仅超级用户](permissions)

Use：`~config write <attribute> <value> [-s]`

Parameter：

-   `-s`：adds configuration information to secret class.

This command can add configuration information.

## `config delete`

-   [仅超级用户](permissions)

Use：`~config delete <Property>`

This command can remove configuration information.
