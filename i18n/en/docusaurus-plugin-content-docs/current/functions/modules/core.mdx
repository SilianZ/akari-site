# core

此模块是小可最基础的模块，可以查看帮助以及使用管理员设置。

## `ping`

-   [公开](permissions)

用法：`~ping`

此命令可以得知机器人是否在线，若拥有超级用户权限则可获取系统信息。

:::info 例子

返回的结果示例：

```
>>> ~ping
<<< Pong!
系统运行时间：2022-12-23 20:31:36
机器人已运行：0:57:33.924041
Python版本：3.9.12
处理器型号：Intel(R) Xeon(R) Platinum 8255C CPU @ 2.50GHz
当前处理器使用率：3.2%
物理内存：7443M 使用率：74.6%
Swap内存：2047M 使用率：99.9%
磁盘容量：43G/196G
```

:::

## `help`

-   [公开](permissions)

用法：`~help [<命令>]`

此命令可以查看机器人所有命令的帮助信息，输入命令可查看其对应的帮助信息。

## `whoami`

-   [公开](permissions)

用法：`~whoami`

此命令可以获取发送命令者在机器人内部的 ID 。

:::info 例子

返回的结果示例：

```
>>> ~whoami
<<< 你的 ID 是：QQ|xxx
本对话的 ID 是：QQ|Group|xxx
```

:::

## `version`

-   [公开](permissions)

用法：`~version`

此命令可以获得目前机器人运行的软件版本。版本号为 Git commit hash 的前六位。

## `mute`

-   [仅管理员](permissions)

用法：`~mute`

此命令可以切换是否允许机器人发言。

:::caution

请勿直接禁言机器人，这是违反[服务条款](https://bot.teahouse.team/wiki/服务条款)的行为，因为这样会提高机器人账号被冻结的风险。若机器人发现自己被禁言则会退出该群聊。

:::

## `leave`

-   [仅管理员](permissions)

-   别名：`dismiss`

用法：`~leave`

此命令可以使机器人离开群聊。

:::caution

请勿直接将机器人移出 QQ 群，这是违反[服务条款](https://bot.teahouse.team/wiki/服务条款)的行为，因为这样会提高机器人账号被冻结的风险。

:::
