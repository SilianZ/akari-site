# mkey

-   [闭源]\(help/faq#Why some modules are not open sources)

This module calculates a panacea password for the custody of hosts in the Range series.

## `mkey`

-   [公开](permissions)

Use：`~mkey <host> <month> <date> <confirmation code> [<host ID>]`

This command calculates the panacea password that is administered by the Protestant Custodian.

Supported hosts：`3ds`, `dsi`, `wii`, `wiiu`, `witch`.

The month and date are the dates set in the host. The confirmation code can be found here (in switch).

![](https://z1.ax1x.com/2023/11/18/piNEJwq.png)

Switch host needs additional host ID.
