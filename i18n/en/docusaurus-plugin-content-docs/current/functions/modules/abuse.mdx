# Abuses

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Force turn on
-   Alias：`ae`

This module is used to manage user[滥用警告](abuse_warning).

## `abuse check`

-   [仅超级用户](permissions)

Use：`~abuse check <User ID>`

This command can check the number of warnings from users.

## `abuse warn`

-   [仅超级用户](permissions)

Use：`~abuse warn <User ID> [<Warning>]`

This command can add user abuse warnings.

## Use revoke\`

-   [仅超级用户](permissions)

Use：`~abuse revoke <User ID> [<Warning>]`

This command removes the number of user abuse warnings.

## `abuse clear`

-   [仅超级用户](permissions)

Use：`~abuse clear <User ID>`

This command can clear the number of user abuse warnings.

## `abuse untempban`

-   [仅超级用户](permissions)

Use：`~abuse unempban <User ID>`

This command can remove the user's temporary blocking.

## `abuse ban`

-   [仅超级用户](permissions)

Use：`~abuse ban <User ID>`

This command can block users.

## `abuse unban`

-   [仅超级用户](permissions)

Use：`~abuse unban <User ID>`

This command can unblock users.
