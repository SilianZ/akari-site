# eval

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Force turn on

This module returns the result of an expression that is passed into a string.Equivalent to the `eval()` function.

Parameters can be changed in[配置文件](development/config) to enable or not.

## `eval`

-   [仅超级用户](permissions)

Use：`~eval <string>`

This command can use the string as a valid expression and return the result.
