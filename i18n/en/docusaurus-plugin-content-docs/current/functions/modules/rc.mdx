# rc

- 依赖：[wiki](wiki)

此模块可以查询设置的起始 Wiki 的最近更改。

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::

## `rc`

- [公开](permissions)
- 转发消息

此命令可以查询最近更改。

用法：`~rc [legacy]`

加入 `legacy` 强制使用文字模式。

:::info 例子

返回的结果示例：

```
>>> ~rc
<<< https://zh.minecraft.wiki/w/Special:RecentChanges
MediaWiki:Conversiontable/zh-tw - MysticNebula70 20:01（UTC+8）
命令/execute - Eason2013 20:00（UTC+8）
Minecraft Wiki:社区专页 - Geticer 19:55（UTC+8）
User talk:JYN2018X - KaplanSteve 19:52（UTC+8）
File:Launcher 2.14.12 Literary.png - Anterdc99 19:52（UTC+8）
…仅显示前 5 条内容。
```

:::
