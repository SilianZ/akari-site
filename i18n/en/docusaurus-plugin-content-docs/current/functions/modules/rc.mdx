# rc

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Depends：[wiki](wiki)

This module will be able to query the latest changes from the set start wiki.

## `rc`

-   [公开](permissions)
-   Forward Message

This command can query the latest changes made by the starting wiki set up.

Use：`~rc`

:::info Example

Result Example：

```
>>>> ~rrc
<<<https://minecraft.fandom>. om/en/wiki/Special: RecentChanges
Minecraft Dungeons: Level - Zyjking 1718 (UTC+8)
Minecraft Dungeons: Flushes Cage - Reverse88 1654 (UTC+8)
Creative Mode - Lone42101 1628 hours (UTC+8)
Minecraft Dungeons: Biological - Zyjking 1620 hours (UTC+8)
Minecraft Dungeons:Squid - Zyjking 16:19 hours (UTC+8)
…The first 5 items are shown.
```

:::

:::warning

Some websites have been blacklisted, and some information on some sites cannot be obtained using this command.Even if the web is not blacklisted, the returned content will still be cloudable and intelligible.Don't die.

:::
