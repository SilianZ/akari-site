# rc

-   [开源](/docs/help/faq#为什么有些模块没有开源)
-   依赖：[wiki](wiki)

此模块可以查询设置的起始 Wiki 的最近更改。

## `rc`

-   [公开](permissions)
-   转发消息

此命令可以查询设置的起始 Wiki 的最近更改。

用法：`~rc`

:::info 例子

返回的结果示例：

```
>>> ~rc
<<< https://minecraft.fandom.com/zh/wiki/Special:RecentChanges
Minecraft Dungeons:等级 - Zyjking 17时18分（UTC+8）
Minecraft Dungeons:刷怪笼 - Reverse88 16时54分（UTC+8）
创造模式 - Lone42101 16时28分（UTC+8）
Minecraft Dungeons:生物 - Zyjking 16时20分（UTC+8）
Minecraft Dungeons:鱿鱼 - Zyjking 16时19分（UTC+8）
…仅显示前 5 条内容。
```

:::

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::
