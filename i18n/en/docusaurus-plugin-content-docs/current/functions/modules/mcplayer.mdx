# mcplayer

-   [开源]\(help/faq#Why some modules are not open sources)

This module can use the Mojang API to search for information about a Minecraft：Java Player.

## `mcplayer`

-   [公开](permissions)

Use：`~mcplayer <Player Name|UUID>`

This command can fetch player information via username or UUUID.

:::info Example

Result Example：

```
>>>> ~mcplayer jeb_
<<<<jeb_(853c80e3c3749fdaaa49938b674adae6)
NameMC：https://namec.com/profile/jeb_
```
