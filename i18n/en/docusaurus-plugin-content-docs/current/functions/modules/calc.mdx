# calc

-   [开源]\(/docs/help/faq#Why some modules are not open sources)

This module relates to the calculator.

All commands in this module will be calculated at a limit of 10 seconds, after which they will be forced to stop.

## `calc`

-   [公开](permissions)

Use：`~calc <mathematical expression>`

This command calculates a mathematical expression.For help on mathematical expression, see[数学表达式](/docs/help/expressions).

This command supports mathematical functions, statistical functions, and plural operations (needs to be added to the prefix `cmath.`).
