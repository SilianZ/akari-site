# locale

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Force turn on

This module controls multilingual related features.

## `locale`

-   [仅管理员](permissions)

Use：`~locale [<language code>]`

This command allows you to view or toggle the worker's language.

## `locale load`

-   [仅超级用户](permissions)

Use：`~locale load`

This command can reload the language file.
