# chemical_code

-   [开源](/docs/help/faq#为什么有些模块没有开源)
-   别名：`cc`、`ccode`

This module generates a chemical structure and requires corresponding molecular formula.

This module was modified from [Chemical_Code](https://github.com/lunarbunnys/Chemical_Code), from [ChemSpider](https://www.chemspider.com/).

:::warning

This command is vulnerable to abuse, do not use too many times in a short period of time or trigger[滥用警告](/docs/abuse_warning).滥用命令是违反[服务条款](policy)的行为，不仅会影响正常交流，还有可能导致机器人账号被风控。

:::

## `chemical_code`

-   [公开](permissions)

Use：`~chemical_code [<ChemSpider ID>]`

This command can generate a structure in which the user needs to answer within a limited time.

Enter the ChemSpider ID to customize the title, enter it as 0 or leave blank to random.Answers use alphabetical order.

If the answer is correct, you can get 1 piece of peal, your custom title won't get a valve.

:::info Example

Result Example：

```
>>> ~chemical_code
<<< ID：236
请在 2 分钟内发送这个化合物的分子式。（除C、H外使用字母表顺序，如：CHBrClF）
```

![](https://s1.ax1x.com/2023/09/02/pPBLVgg.png)

Correct answer sample：

```
>>>> C6H6
<<< Correct Answers.
```

Timeout failed to answer sample：

```
<<< Timeout, the correct answer is C6H6.
```

:::

## `chemical_code captcha`

-   [公开](permissions)
-   Alias：`ccaptcha`

Use：`~chemical_captcha`

Standard verification code style.The timer system was removed and the first message sent by the initiator was answered and the custom title is not supported.

Get 2 pieces of valve valve if the answer is correct.

:::info Example

Result Example：

```
>>> ~chemical_code captcha
<<< ID：236
请发送这个化合物的分子式。（除C、H外使用字母表顺序，如：CHBrClF）
```

![](https://s1.ax1x.com/2023/09/02/pPBLVgg.png)

Correct answer sample：

```
>>>> C6H6
<<< Correct Answers.
```

Incorrect answer to sample：

```
<<<< Answer Error, the correct answer is C6H6.
```

:::

## `chemical_code stop`

-   [公开](permissions)

Use：`~chemical_code stop`

This command will stop when the module is running.
