# reverse

-   [开源]\(help/faq#Why some modules are not open sources)
-   Alias：`conv`, `unit`

This module can convert units.This module supports most unit conversions.

This module is based on [Pint](https://github.com/hgrecco/pint).

## `convert`

-   [公开](permissions)

Use：`~convert<pre-conversion values> <rear unit>`

This command can use the Pint conversion unit.Unit symbols are case-sensitive.

:::info Example

Result Example：

```
>>> ~convert km mi
<<<1 km = 0.6213711922373339696174341841 mi
```

:::
