# idlist

-   [开源]\(/docs/help/faq#Why some modules are not open sources)

This module makes it possible to find the base rock ID.

Source from[命令助手](https://ca.projectxero.top/).

## `idlist`

-   [公开](permissions)

Use：`~idlist <description> [<filters>...]`

This command can query the MCBEID table.
