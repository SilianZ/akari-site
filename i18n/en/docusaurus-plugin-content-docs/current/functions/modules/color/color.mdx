# color

-   Alias：`colour`

This module provides color information.

## `color`

-   [公开](permissions)

Use：`~color [<color>]`

This command gives colors information about the colors.

This command supports the following color format：

-   16 Intraction (#`omitted)：`#123456`, #123`
-   RGB：`rgb(12,34,56)`
-   HSL：`hsl(123,45%,67%)`
-   CSS color ([列表](color_list#CSS))：`blue`
-   Material Design Colors ([列表](color_list#material-design)：`blue-500`

Leave empty to random color.

If the color exists in the CSS or Material Design color, the corresponding color code will be displayed in the image.

:::info Example

Result Example：

![](https://z1.ax1x.com/2023/11/06/pilBEkQ.png)

```
>>>> ~color #0091ff
<<< HEX：#0091ff
RGB：rgb(0, 145, 255)
HSL：hsl(205, 100%, 50%)
The most similar CSS color name: dodgerblue
The most similar material Design color name: lightblue-A700
```

:::
