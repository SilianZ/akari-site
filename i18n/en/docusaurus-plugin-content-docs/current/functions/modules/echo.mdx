# echo

-   [开源]\(help/faq#Why some modules are not open sources)
-   Force turn on

This module allows robots to repeat messages.

## `echo`

-   [仅超级用户](permissions)

Use：`~echo <message>`

This command can duplicate messages.

## `say`

-   [仅超级用户](permissions)

Use：`~say <message>`

This command can repeat messages without responding.

:::info

The message sent by the robot itself does not trigger its own command.

:::

## `raise`

-   [仅超级用户](permissions)

This command can throw exception to test it.
