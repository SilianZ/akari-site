# help

-   [开源]\(/docs/help/faq#Why some modules are not open sources)

This module allows you to view command help.

## `help`

-   [公开](/docs/permissions)
-   Force turn on

Use：`~help [<command>]`

This command can view the help of all robot commands, enter the command to see its corresponding help information.

## `help legacy`

-   [公开](/docs/permissions)

Use：`~help legacy`

This command can force the help to be viewed in text mode.
