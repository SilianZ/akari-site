# ip

-   [开源]\(/docs/help/faq#Why some modules are not open sources)

This module can query IP information.

## `ip`

-   [公开](permissions)

Use：`~ip <IP>`

This command can query IP information.

:::info Example

Result Example：

```
>>> ~ip 8.8.8.8
<<< 8.8.8.8
Type：IPv4 Global Address
Position：Los Angeles, California, United States (-118.2441°W, 34. 544°N)
Zip code：90009
Organization：Google
ASN：15169 (GOGLE)
Timezone：UTC-8
Reverse parsing：dns.google
```

:::
