# setup

-   [开源]\(help/faq#Why some modules are not open sources)
-   Force turn on

This module sets the behavior of robots in the group.

## `setup typeping`

-   [仅管理员](permissions)
-   Settings

Use：`~setup typeping`

This command can toggle whether or not to display input prompts.

Show as a stamp on QQ and as "typing..." on Discord

## `setup check`

-   [仅管理员](permissions)
-   Settings

Use：`~setup check`

This command can toggle whether or not to enable command error check prompts.

## `setup timeoffset`

-   [仅管理员](permissions)
-   Settings

Use：`~setup timeoffset <Offset >`

This command can set the time offset value for the current group.
