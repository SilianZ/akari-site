# setup

-   [开源](help/faq#为什么有些模块没有开源)
-   强制开启

此模块可以设置机器人在群组内的行为。

## `setup typing`

-   [仅管理员](permissions)
-   设置

用法：`~setup typing`

此命令可以切换是否显示输入提示。

在 QQ 显示为戳一戳，在 Discord 上显示为“正在输入...”

## `setup check`

-   [仅管理员](permissions)
-   设置

用法：`~setup check`

此命令可以切换是否开启命令错字检查提示。

## `setup timeoffset`

-   [仅管理员](permissions)
-   设置

用法：`~setup timeoffset <偏移值>`

此命令可以设置当前群组的时间偏移值。
