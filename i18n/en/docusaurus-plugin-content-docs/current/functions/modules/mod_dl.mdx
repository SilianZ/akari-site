# mod_dl

此模块可以下载 Curseforge 和 Modrinth 上的 Mod。

## `mod_dl`

- [公开](permissions)

用法：`~mod_dl <Mod 名称> [<版本>]`

通过 Mod 名称获取下载链接。若不事先指定版本则会在搜索时确定版本。
