# err

-   [开源]\(help/faq#Why some modules are not open sources)

This module will be able to query details of the error number of the host series of the Parameter.

## `err`

-   [公开](permissions)

Use：`~err <Error Code>`

:::info Example

This is a sample of returned results：

```
2181-4008/0x1f50b5 (Nintendo Switch)
Module:
dauth (181)
Description:
Console is permanently attached by Nintendo. (4008)
Further information:
https://en-americas-support.nintendo.com/app/answers/detail/a_id/42061/kw/2181-4008
```

:::
