# err

此模块可以查询任天堂主机系列报错码的详细信息。

## `err`

- [公开](permissions)

用法：`~err <错误代码>`

:::info 例子

这是返回的结果示例：

```
2181-4008/0x1f50b5 (Nintendo Switch)
Module:
dauth (181)
Description:
Console is permanently banned by Nintendo. (4008)
Further information:
https://en-americas-support.nintendo.com/app/answers/detail/a_id/42061/kw/2181-4008
```

:::
