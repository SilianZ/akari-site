# whois

-   [开源]\(help/faq#Why some modules are not open sources)

This module makes it possible to query WHOIS for domain names.

## `whois`

-   [公开](permissions)

Use：`~whis <DNS >`

This command can query WHOIS.

:::info Example

Result Example：

```
>>>> ~whois google.com
<<<<< domain name：google.com
registrants：MarkMonitor, Inc.
WHOIS server：whois.markmonitor. om
Updated：2019-09-09 15:39:04
Registration date：1997-09-15 04:00:00
Due date：2028-09-14 04:00:00
DNS server：ns1. oogle.com, ns3.google.com, ns2.google.com, ns4.google.com
DNSSEC：unsigned
Organization：Google LC
Position：CA, US
```

:::
