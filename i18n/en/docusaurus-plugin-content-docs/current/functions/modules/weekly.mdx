# weekly

-   [开源]\(/docs/help/faq#Why some modules are not open sources)

This module can fetch the Chinese Minecraft Wiki weekly or Tea Gallery Week.

## `weekly`

-   [公开](permissions)

Use：`~weekly`

This module fetches the Chinese Minecraft Wiki weekly page.

## `weekly teahouse`

-   [公开](permissions)

Use：`~weekly teahouse`

This module can get Tea Weekly Bulletin.
