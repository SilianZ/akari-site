# mcv

-   [开源]\(/docs/help/faq#Why some modules are not open sources)

This module can query the latest Minecraft version.

## `mcv`

-   [公开](permissions)
-   Alias：`m`

Use：`~mcv`

This command can query the latest official, snapshot version of Minecraft：Java launcher and the latest formal, snapshot version of Mojira (Mojang Studios's bug reporter).

## `mcbv`

-   [公开](permissions)

Use：`~mcbv`

This command can query the latest version of Minecraft rock recorded in Mojira (Mojang Studios's Bug Reporter).

## `mcdv`

-   [公开](permissions)

Use：`~mcdv`

This command can query the latest version of Minecraft Dungeons in Mojira (Mojang Studios's Bug Reporter).

## `mcev`

-   [公开](permissions)

Use：`~mcev`

This command can query the latest version of the Windows version of Minecraft：education.
