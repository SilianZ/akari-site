# b

-   [开源]\(/docs/help/faq#Why some modules are not open sources)

This module can query the abuse log of the starting wiki set.

## `ab`

-   [公开](/docs/permissions)

This command can query the abuse log of the starting wiki set.

Use：`~ab`
