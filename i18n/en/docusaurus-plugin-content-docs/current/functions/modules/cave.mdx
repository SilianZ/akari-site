# cave

-   [开源]\(help/faq#Why some modules are not open sources)

This module can use the echo hole.

Replication hole resembles a message board, users can store their own messages.

## `cave`

-   [公开](permissions)
-   Cool 90 seconds

Usage：

-   `~cave [<id>]`

This command allows you to fetch the echo hole.

Enter an id to specify a return hole number.

:::info Example

Result Example：

```
>>>> ~cave
<<<< Back Cave #33
Munde
— Emerald-AM9
```

:::

## `save add`

-   [公开](permissions)

Usage：

-   `~save add`

This command can add a sound hole.

## `save remove`

Usage：

-   `~save remove <id>`
-   `~cave dedebele <id>`

This command can remove the sound hole of the given id.
