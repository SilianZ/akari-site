# twenty_four

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Alias：`24`

This module can play 24 points.

:::warning

This command is susceptible to abuse, please do not use too many times in a short period of time or trigger[滥用警告](abuse_warning).Abuse of command is a violation of[服务条款](policy), which not only affects normal communication, but can also cause robot accounts to be controlled.

:::

## `twenty_sour`

-   [公开](permissions)

Use：`~twenty_sour`

This command can generate 4 numbers. Requires the initiator to use all numbers to form a mathematical expression with a result of 24.If a digital combination is unsolved, it can be sent 'Unsolve'.

If the answer is correct, you can get 2 pieces of peal, "Unsolved" answer error, you lose 1 piece of peal.

Supported operators have `+`, `-`, `*', `/`and`()\`. For help on mathematical expressions, see[数学表达式](help/expressions).

:::info Example

Result Example：

```
>>>> ~twenty_four
<<<< assigned digital combination：[1, 3, 5, 7]
Please enter an expression to make it 24. (If unsolved, enter 'Unsolve')
```

Correct answer sample：

```
>>>> (1+5)*(7-3)
<<< Correct Answers.
```

Error answer sample：

```
<<<< Error in answer.
```

:::
