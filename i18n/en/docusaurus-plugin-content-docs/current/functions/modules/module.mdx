# Module

-   [开源]\(help/faq#Why some modules are not open sources)
-   Force turn on

This module can be enabled or disabled.

## `module enable`

-   [仅管理员](permissions)
-   Alias：`enable`

Usage：

-   Specify：\`~module enabling \<Module 1> [<Module 2>...] [-g]
-   All：`~module enables all [-g]`

Parameter：

-   `-g`：applies to the channel globally.

This command can turn on specified or all modules.

## `module disable`

-   [仅管理员](permissions)
-   Alias：`disable`

Usage：

-   Specify：\`~module disable\<Module 1> [<Module 2>...] [-g]
-   All：`~module disable all [-g]`

Parameter：

-   `-g`：applies to the channel globally.

This command can close the specified or all modules.

## `module list`

-   [仅管理员](permissions)

Use：`~module list`

This command can list all modules of the current robot.

## `module list legacy`

-   [仅管理员](permissions)

Use：`~module list legacy`

This command can force text mode to list all modules of the current robot.

## `module load`

-   [仅超级用户](permissions)
-   Alias：`load`

Use：`~module load <1> [<Module 2>...]`

This command can reload the module.

## `module unload`

-   [仅超级用户](permissions)
-   Alias：`unload`

Use：`~module unload<Module 1> [<Module 2>...]`

This command can uninstall the module and leave it unloaded on reboot.

## `module load`

-   [仅超级用户](permissions)
-   Alias：`load`

Use：`~module load <Module 1> [<Module 2>...]`

This command can load uninstalled modules.
