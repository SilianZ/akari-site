# 模块

小可提供了一部分的官方模块，可使用 `~module` 命令对模块进行操作，但基础模块不会被此命令影响。

若您准备[自搭建](/docs/dev/deploy)机器人，请注意并非所有模块为立即可用，部分模块需要修改[配置文件](/docs/dev/config)或进行额外操作才能正常使用。

您可以编写自定义模块，请将其放到 `modules` 目录下，可以在[新建模块指南](/docs/dev/new_module)获得更多信息。

您也可以在 Issue 中[请求新功能](https://github.com/Teahouse-Studios/akari-bot/issues/new?assignees=OasisAkari&labels=enhancement&projects=&template=feature_request.yaml&title=[FEATURE]%3A+)，但存在包括但不限于以下情况的模块可能会被搁置：

-   实用性较低
-   服务稳定性不足
-   易消耗机器人性能
-   使用非开源或开源协议不兼容的代码
