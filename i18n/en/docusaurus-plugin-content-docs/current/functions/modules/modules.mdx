# Modules

A small part of the official module can be provided, using the `~module` command to operate on the module, but the base module will not be affected by this command.

If you prepare the[自搭建](/docs/development/employment) robot, please note that not all modules are immediately available, some modules need to modify[配置文件](/docs/development/config) or make additional actions to work properly.

You can write a custom module. Make it a `modules` directory and get more information at[新建模块指南](/docs/development/new_module).

您也可以在 Issue 中[请求新功能](https://github.com/Teahouse-Studios/akari-bot/issues/new?assignees=OasisAkari&labels=enhancement&projects=&template=feature_request.yaml&title=[FEATURE]%3A+)，但存在包括但不限于以下情况的模块可能会被搁置：

-   Low utility
-   Inadequate stability of services
-   Easily consume robot performance
-   Use non-open source or open source protocol incompatible code
