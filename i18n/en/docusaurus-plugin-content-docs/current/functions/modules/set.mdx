# set

-   [开源]\(help/faq#Why some modules are not open sources)

This module can operate the robot database.

## `set modules`

-   [仅超级用户](permissions)

Use：`~set modules <Target ID> <Modules>`

This command can configure a module for an object.

## `set option`

-   [仅超级用户](permissions)

Use：`~set option <Target ID> <Keyname> <Value>`

This command can set parameters for an object.

## `post_whitelist`

-   [仅超级用户](permissions)

Use：`~post_whiteist <group ID>`

This command can set a whitelist of the QQ group pushs.
