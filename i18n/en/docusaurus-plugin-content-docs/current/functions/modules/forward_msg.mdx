# forward_msg

-   [开源]\(help/faq#Why some modules are not open sources)

This module will switch on whether to enable forward messages.

## `forward_msg`

-   [仅超级用户](permissions)

Use：`~forward_msg`

This command can toggle whether forward messages are enabled (e.g. `rc`).
