# toggle

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Force turn on

This module can set robot behavior.

## `toggle typeping`

-   [仅管理员](permissions)
-   [设置]\(/docs/help/reading#command type)

Use：`~toggle typeing`

This command can toggle whether or not to display input prompts.

Show as a stamp on QQ and as "typing..." on Discord

## `toggle check`

-   [仅管理员](permissions)
-   [设置]\(/docs/help/reading#command type)

Use：`~toggle check`

This command can toggle whether or not to enable command error check prompts.
