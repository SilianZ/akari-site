# toggle

-   [开源](/docs/help/faq#为什么有些模块没有开源)
-   强制开启

此模块可以设置机器人行为。

## `toggle typing`

-   [仅管理员](permissions)
-   [设置](/docs/help/reading#命令类型)

用法：`~toggle typing`

此命令可以切换是否显示输入提示。

在 QQ 显示为戳一戳，在 Discord 上显示为“正在输入...”

## `toggle check`

-   [仅管理员](permissions)
-   [设置](/docs/help/reading#命令类型)

用法：`~toggle check`

此命令可以切换是否开启命令错字检查提示。
