# server

-   [开源]\(help/faq#Why some modules are not open sources)
-   Alias：`s`

This module will be able to query information about a Minecraft server.

## `server`

-   [公开](permissions)

Use：`~server <server IP or domain name>[:<port>] [-r] [-p]`

Parameter：

-   `-r`：shows original messages
-   \``-p`：Show Players List

This command can query information about a Minecraft server.

:::info Example

Result Example：

General：

```
>>> ~server hypixel. et
<<<< [JE]
                Hypixel Network [1.8-1.19]
            3RD RD SKYBLOCK ANNIVERSARY
Online player：47649/200000
Game version：Requires MC 1. 1.19
Hypixel.net:25565
```

Raw message：

```
>>> ~server hypixel. et -r
<<<< [JE]
                §aHypixel Network §c[1.8-1.19]
            §d§l3RD SKYBLOCK ANNIVERSARY
Online player：47666/200000
Game Version：Requires MC 1. 1.19
Hypixel.net:25565
```

Players list：

```
>>> ~server hypixel. et -p
<<<< [JE]
                Hypixel Network [1.8-1.19]
            3RD RD SKYBLOCK ANNIVERSARY
Online player：47636 / 200000
Current online player：
(Low)
Game version：Requires MC 1. 1.19
Hypixel.net:25565
```

:::
