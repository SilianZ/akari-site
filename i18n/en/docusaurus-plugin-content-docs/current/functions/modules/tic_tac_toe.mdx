# tic_tac_toe

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Alias：`ttt`

This module can play tickets.

Enter a number at the start to represent the selection of the number of cells and send a single 1-9 number or two 1-3 numbers (respectively for lines and columns).

:::warning

This command is susceptible to abuse, please do not use too many times in a short period of time or trigger[滥用警告](abuse_warning).Abuse of command is a violation of[服务条款](policy), which not only affects normal communication, but can also cause robot accounts to be controlled.

:::

## `tic_tac_toe`

-   [公开](permissions)

Use：`~tic_tac_toe`

This command can play bots with robots. Robots may fail.

Win you get 1 piece of flower valve.

## `tic_tac_toe expert`

-   [公开](permissions)

Use：`~tic_tac_toe expert`

This command can play with expert robots. Robots occasionally fail.

Win you get 2 pieces of flower valve.

## `tic_tac_toe master`

-   [公开](permissions)

Use：`~tic_tac_toe master`

This command can play with master robots. The robot will not fail, so you won't win.

## `tic_tac_toe noob`

-   [公开](permissions)

Use：`~tic_tac_toe noob`

This command can play with the bird robot and the robot will always fail.

## `tic_tac_toe duo`

-   [公开](permissions)

Use：`~tic_tac_toe duo`

This command can play with two people

## `tic_tac_toe stop`

-   [公开](permissions)

Use：`~tic_tac_toe stop`

This command will stop when the module is running.
