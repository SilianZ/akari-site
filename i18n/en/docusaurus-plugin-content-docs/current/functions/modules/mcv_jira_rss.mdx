# mcv_jira_rss

-   [开源]\(help/faq#Why some modules are not open sources)

This module can send a notification when a new Minecraft version is updated in Mojira (Minecraft Studio - Mojang Studios).

## mcv_jira_rss

-   [订阅]\(/docs/help/reading#command type)

This module can be notified when Mojira has a new Minecraft：Java update.

::info Example

This is an example of a message sent by the robot when the version is updated：

```
Mojira updated 21w05b.
(information on Mojira is only used for version preview, not for launcher updating)
```

:::

## mcbv_jira_rss

-   [订阅]\(/docs/help/reading#command type)
