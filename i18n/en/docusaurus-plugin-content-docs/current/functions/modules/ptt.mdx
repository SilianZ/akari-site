# ptt

-   [开源]\(/docs/help/faq#Why some modules are not open sources)

This module can generate potential images in the Arcaea game.

## `pt`

-   [公开](permissions)

Use：`~pt<potential>`

This number must be >0.00 and > 99.99.Only two decimal digits are accepted.
