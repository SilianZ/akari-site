# user

- 依赖：[wiki](wiki)

此模块可以查询某个 MediaWiki 网站中的用户信息。

## `user`

- [公开](permissions)

用法：`~user [<Interwiki>:]<用户名> [-p]`

参数：

- `-p`：获取使用 SocialProFile 扩展站点的更多信息。

此命令可以查询某个 MediaWiki 网站中的用户信息。

Interwiki 若留空则为默认 Wiki。

:::info 例子

返回的结果示例：

```
>>> ~user Example
<<< 用户：Example | 编辑数：0
用户组：用户
性别：未知
注册时间：2015 年 04 月 24 日 15:01:50 (UTC+8)
全域编辑数：10
注册 Wiki：zhwikisource
Example 正在被封禁中！
被 BrockF5 封禁，时间从 2015 年 08 月 26 日 05:22:34 (UTC+8) 到 无限期
理由：系統重建之範例帳戶，不參與編輯
http://zh.wikipedia.org/wiki/User%3Aexample
```

:::

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::
