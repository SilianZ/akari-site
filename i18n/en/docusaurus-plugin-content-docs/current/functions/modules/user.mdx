# user

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Alias：`u`

This module provides access to user information on a MediaWiki website.

## `user`

-   [公开](permissions)

This command can query user information on a MediaWiki website.

Use：`~user [-<p|r>] [~<Gamepedia 网站域名>] [<interwiki>:]<Username>`

Parameter：

-   `-p`：returns an image of a user in Gamepedia and only the number of edits will be displayed in a picture other than Gamepedia user.
-   `-r`：returns more information from Gamepedia.
