# github

-   [开源](/docs/help/faq#为什么有些模块没有开源)
-   别名：`gh`

This module can search for Github for specified repo(store) or user details.

:::warning

Some websites have been blacklisted, and some information on some sites cannot be obtained using this command.Even if the web is not blacklisted, the returned content will still be cloudable and intelligible.Don't die.

:::

## `github`

-   [公开](permissions)

Use：`~github <repository or user>`

This command attempts to automatically identify and distinguish between repository and user.

## `github po`

-   [公开](permissions)

Use：`~github repo <repository >`

This command can query the specified repository details for Github.

:::info Example

Result Example：

```
>>> ~github repo Teahouse-Studios/akari-bot

License: AGPL-3.

Website: https://bot.teahouse.team
https://github.com/Teahouse-Studios/akari-bot
```

## `github search`

-   [公开](permissions)

Use：`~githubb search <keyword>`

This command can search the repository on GitHub.

:::info Example

Result Example：

```

<<< 搜索成功，共 6 条结果：
```
