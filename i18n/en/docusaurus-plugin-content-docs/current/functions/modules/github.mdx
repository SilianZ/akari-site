# github

- 别名：`gh`

此模块可以查询 Github 的指定 repo（仓库）或用户详情。

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::

## `github`

- [公开](permissions)

用法：`~github <储存库或用户>`

此命令会尝试自动识别并区分储存库或用户。

## `github repo`

- [公开](permissions)

用法：`~github repo <储存库>`

此命令可以查询 Github 的指定储存库详情。

:::info 例子

返回的结果示例：

```
>>> ~github repo Teahouse-Studios/akari-bot
<<< Teahouse-Studios/akari-bot (272003394)
茶馆群内QQ机器人（小可）

Language · Python | Fork · 24 | Star · 125 | Watch · 125
License: AGPL-3.0
Created 1086 day(s) ago | Updated 9 day(s) ago

Website: https://bot.teahouse.team
https://github.com/Teahouse-Studios/akari-bot
```

:::

## `github user`

- [公开](permissions)

用法：`~github user <name>`

此命令可以查询 Github 的指定用户或组织详情。

## `github search`

- [公开](permissions)

用法：`~github search <关键词>`

此命令可以搜索 GitHub 上的储存库。

:::info 例子

返回的结果示例：

```
>>> ~github search teahouse-studios
<<< 搜索成功，共 6 条结果：
Teahouse-Studios/teahouse-studios.github.io: https://github.com/Teahouse-Studios/teahouse-studios.github.io

另有 1 条结果未显示。
```

:::
