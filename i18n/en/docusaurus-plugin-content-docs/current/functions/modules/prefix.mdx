# Prefix

-   [开源]\(help/faq#Why some modules are not open sources)
-   Force turn on

This module can customize the prefix for command.

## `Prefix add`

-   [仅管理员](permissions)

Use：`~prefix add <prefix >`

This command can set custom robot command prefix.

## `prefix remove`

-   [仅管理员](permissions)

Use：`~prefix remove <prefix >`

This command can remove custom robot prefix.

## `prefix reset`

-   [仅管理员](permissions)

Use：`~prefix reset`

This command can reset custom robot command prefix.

## `prefix list`

-   [仅管理员](permissions)

Use：`~prefix list`

This command can list the custom command prefix.
