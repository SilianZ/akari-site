# bugtracker

-   [开源]\(help/faq#Why some modules are not open sources)
-   Alias：`bug`

This module will find details about a specific bug report from Mojira (Minecraft Development Studio Mojang Studios).

## `bugtracker`

-   [公开](permissions)

Usage：

-   `~bugtracker <bug ID>`
-   Direct bug ID：`<bug ID>`
-   Use URL：`HTML[s]http://bugs.mojang.com/browse/<bug ID>`

This command can ask for details of a specific bug report from Mojira.

This command cannot search for privacy holes.

:::info Example

Result Example：

```
>>>> ~bugtracker MC-4
<<< [MC-4] Item drops somtimes appear at the wrong location
Type: Bug
Project: Minecraft: Java Edition
Status: Reopened
Mojang Priority: Low
Resolution: Unresolved
Versions: Minecraft 1. 1 ~1.20 Pre-release 6
https://bugs.mojang.com/browse/MC-4
```

:::
