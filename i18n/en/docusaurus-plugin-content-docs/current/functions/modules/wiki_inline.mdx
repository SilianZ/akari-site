# wiki_inline

- 依赖：[wiki](wiki)

此模块可以让你无需机器人前缀查询一个基于 MediaWiki 的网站的指定页面的详情。

## `wiki_inline`

- [公开](permissions)
- 别名：`wiki_regex`

用法：

- 查询页面：`[[[<interwiki>]:<页面名>]]]`
- 查询模板：`{{<模板名>}}`
- 查询 MediaWiki 命名空间的页面：`≺<页面名>≻`或`⧼<页面名>⧽`

此命令可以让你无需机器人前缀查询一个基于 MediaWiki 的网站的指定页面的详情。

![](https://z1.ax1x.com/2023/11/18/piN0Tkq.png)

（返回 Infobox 示例）

若页面中存在 Infobox，则会渲染并展示页面中的 Infobox 图像。

:::tip

注意：上方用法中，前两个方括号是实际需要输入的，而第三个括号则是用于表示可选参数的。例如：

- `[[mcwiki:苦力怕]]`
- ~~`[[[mcwiki:苦力怕]]`~~

:::

:::tip 无法使用？

请检查是否参照 [wiki](wiki) 进行配置。

:::

:::warning

此命令易被滥用，请勿在短时间内使用太多次，否则会触发[滥用警告](abuse_warning)。滥用命令是违反[服务条款](policy)的行为，不仅会影响正常交流，还有可能导致机器人账号被风控。

:::

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::
