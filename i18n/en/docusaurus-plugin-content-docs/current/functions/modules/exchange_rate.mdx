# exchange_rate

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Alias：`excr`

This module can query currency exchange rates and convert currency prices.

Source from [ExchangeRate-API](https://www.exchangerate-api.com/).

## `exchange_rate`

-   [公开](permissions)

Usage：

-   `~exchange_rate <value+original currency code > <Convert currency code>`
-   `[<value>]<original currency code>(real|change)<Convert currency code>`

This command can convert prices between different currencies, exchange rate data is updated every 24 hours.

If the amount is left blank, default is 1.

:::info Example

Result Example：

```
>>>> ~exchange_rate 100CNY USD
<<<< 100.0 CNY = 13.93 USD
(2023-06-21, for reference only)
```

:::
