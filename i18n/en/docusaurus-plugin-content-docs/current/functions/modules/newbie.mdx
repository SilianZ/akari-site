## `newbie`

-   [公开](permissions)

此命令可以查询设置的起始Wiki的新用户日志。

用法：`~newbie`

:::info 例子

本命令返回的结果示例：

```
>>> ~newbie
<<< https://minecraft.fandom.com/zh/wiki/Special:Log?type=newusers
User:Limengzestudio
User:Miao Siyuan China
User:Jeaaaaan
User:Gill0502
User:Wjxhzs
…仅显示前 5 条内容。
```

:::

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::
