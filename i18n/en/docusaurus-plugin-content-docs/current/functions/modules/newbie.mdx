# newbie

-   [开源]\(help/faq#Why some modules are not open sources)
-   Depends：[wiki](wiki)

This command can query new user logs for the starting wiki set.

:::warning

Some websites have been blacklisted, and some information on some sites cannot be obtained using this command.Even if the web is not blacklisted, the returned content will still be cloudable and intelligible.Don't die.

:::

## `newbie`

-   [公开](permissions)

This command can query new user logs for the starting wiki set.

Use：`~newbie`

:::info Example

Results sample： returned by this command

```
>>>> ~newbie
<<<https://en.minecraft.wiki/w/Special:Log?type=newusers
User:Yuuk4
User:Yxx
User:Sodiums
User:Shigz007
User:HLXF
…The first 5 items are shown only.
```

:::
