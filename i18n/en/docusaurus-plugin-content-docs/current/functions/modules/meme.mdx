This module can be used to query infarctions or abbreviations.

## `me`

-   [公开](permissions)

Use：`~mee<query content>`

This command can search for obfuscations (obstructions) or abbreviations in brides Encyclopaedia, nnhsh, and Urban Dictionary.

:::info Example

Result Example：

```
>>> ~meme LOL
<<< [萌娘百科] 《英雄联盟》（英文：League of Legends，简称LOL、撸啊撸、101），是由Riot Games开发及发行的一款在Windows和OS X平台下的MOBA游戏，中国大陆地区由腾讯游戏代理发行，仅支持Windows平台。
https://zh.moegirl.org.cn/index.php?curid=34898
[nbnhhsh]（2个结果，已收录）：大笑、英雄联盟
[Urban Dictionary]（10个结果）：
lol
[An abbreviation] that stands for 'laugh out loud' to you, but 'lots of [love'] for [your mom].
Example: Mom: [Grandma] just [died]. LOL
Son: Mom [wtf]?
http://lol.urbanup.com/11482563
```
