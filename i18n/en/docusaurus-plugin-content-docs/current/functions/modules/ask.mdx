# ask

此模块可以使用 ChatGPT，使用此模块将消耗约 3 个花瓣。

此模块由 GPT-3.5-turbo-0613 提供支持。

## `ask`

- [公开](permissions)
- 冷却 60 秒

用法：

- `~ask <问题>`
- `问：<问题>？`

此命令可使用 ChatGPT 回答问题。

:::warning

某些网站已经被列入黑名单，使用此命令无法获取某些网站的某些信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤。请不要作死。

:::
