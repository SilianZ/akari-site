# restart

-   [开源](help/faq#为什么有些模块没有开源)
-   强制开启

此模块可以更新机器人。

## `restart`

-   [仅超级用户](permissions)

用法：`~restart`

此命令可以重启机器人软件。

## `update`

-   [仅超级用户](permissions)

用法：`~update`

此命令可以从 GitHub 下载新版本小可并更新。

## `update&restart`

-   [仅超级用户](permissions)
-   别名：`u&r`

用法：`~update&restart`

此命令可以从 GitHub 下载新版本小可并更新，之后重启机器人软件。
