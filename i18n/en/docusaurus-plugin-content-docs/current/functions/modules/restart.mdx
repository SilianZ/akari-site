# start

-   [开源]\(help/faq#Why some modules are not open sources)
-   Force turn on

This module can update robots.

## `Restart`

-   [仅超级用户](permissions)

Use：`~return`

This command can restart robot software.

## `update`

-   [仅超级用户](permissions)

Use：`~update`

This command can download a new version from GitHub and update it.

## `update&restore`

-   [仅超级用户](permissions)
-   Alias：`u&r`

Use：`~update&restore`

This command can download a new version from GitHub and update it, and then restart robot software.
