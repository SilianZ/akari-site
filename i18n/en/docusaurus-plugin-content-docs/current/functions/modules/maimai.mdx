# maimai

-   [闭源]\(/docs/help/faq#Why some modules are not open sources)
-   Alias：`mai`

This module can query information from the Maimai game.

This module was transplanted from [Maibot](https://github.com/Diving-Fish/mai-bot).

If not specified, the difficulty of this page supports full title, three-letter abbreviations and Chinese (Green Yellow Red Purple), letter-size is not sensitive.

In order to prevent normal communication, the results are in the form of pictures for more than 10 items and not for more than 200 items.

:::warning

The data source used for this module commands is[查分器]for Diving-Fish (https://www.diving-fish.com/maimaidx/prober/), please check your registration number and set it before using this command for the first time.

:::

## `maimai base`

-   [公开](permissions)

Use：`~maimai base <number > [<maximum>]`

This command can search for songs according to a certain number.

## `maimai level`

-   [公开](permissions)

Use：`~maimai level <level>`

This command can search for songs according to hierarchy.

Use this command to show more than 200 results normally.

## `maimai search`

-   [公开](permissions)

Use：`~maimai search <search term>`

This command can search for songs based on song name (or partly).

## `maimai alias`

-   [公开](permissions)

Use：\`~maimai alias <ID>

This command can query alias names of songs.The song alias library is provided by Xray

## `maimai info`

-   [公开](permissions)

Use：`~maimai info <ID 或别名> [<detector username>]`

This command can query song's beatmap grade by ID or alias.

ID needs to be added to prefix 'id'. Use underscore '\_' instead of spaces.

Leave blank to default to the username corresponding to the QQ number.

:::info Example

Result Example：

```

<<< 834. PANDORA PARADOXXX Basic 7+
100.7275 SSS+
Advanced 11
100. 481 SSS+ FC
Expert 13+
100.4628 SSS FC
Master 14+
98.8164 S+
Re: MASTER 15
97.5142 S
```

:::

:::tip Unusable?

Please check if you have entered **the identifier** (non-game name or nickname).

:::

## `maimai process`

-   [公开](permissions)

Use：`~maimai process <Rank > <Destination > [<Charger Username>]`

This command can see how far players have completed their ratings at the corresponding level.

Leave blank to default to the username corresponding to the QQ number.

Supported target ratings：

-   Level：`A`, `AAA`, `S`, `S+`, `SS+`, `SSS`, `SSS+`
-   Strike：`FC`, `FC+`, `AP`, `AP+`
-   Sync rate：`FS`, `FS+`, `FDX`, `FDX+`

:::tip Unusable?

Please check if you have entered **the identifier** (non-game name or nickname).

:::

## `maimai song`

-   [公开](permissions)

Use：`~aimai song <ID 或别名> [<difficulty>]`

This command can query song or beatmap information by ID or alias.

ID needs to be added to prefix 'id'. Use underscore '\_' instead of spaces.

:::info Example

Result Example：

Song：

```
>>> ~maimai song id834
<<< 834. PANDORA PARADOXXX
艺术家：削除
分类：舞萌
BPM：150
版本：maimai FiNALE
难度：7.7/11.5/13.9/14.9/15.0
```

Beatmap：

```
>>> ~maimai song id834 Re:MASTER
<<< 834. PANDORA PAADOXXX
Re:MASTER 15(15. )
TAP：1165
HOLD：72
SLIDE：90
BREAK：15
Spectror：PANDORA PARADOXXX
```

:::

## `maimai random`

-   [公开](permissions)

Usage：

-   Specify the：`~maimai random <rating + difficulty> [<type>]`

-   Fully random：`~maimai random`

This command can randomly specify a song with a condition or completely random if no parameters are entered

Enter `#Skipping` if you do not want to specify difficulty and hierarchy, no space is required between difficulty and hierarchy.

Type can be entered either `SD` or `DX`, left empty or else not filtered.

## `maimai scoreline`

-   [公开](permissions)

Use：`~maimai scoreline <ID> <difficulty > <score lines>`

This command can find a curve score line.

The command returns the TAP GREAT tolerance error and the TAP GREAT number of BREAK 2500 or BREAK 2000 equivalents.

:::info Example

Result Example：

```
>>>> ~maimai scoreline 834 Re:MASTER 100
<<< PANDOORA PAADOXXR Re:MASTER
Spectrum 100.0% allowed up to 82.70 (per -0). 121%),
BREAK 2550 (total 15) equal to 1.378 TAP GREAT (-0.0167%),
BREAK 2000 (total 15) equal to 8.38 TAP GREAT (-0.1005%).
```

:::
