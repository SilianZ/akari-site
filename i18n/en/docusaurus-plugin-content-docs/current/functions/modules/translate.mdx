# translate

- 别名：`trans`

此模块可以翻译文本。

本模块基于[百度翻译 API](https://fanyi-api.baidu.com/) 实现。

## `convert`

- [公开](permissions)

用法：`~convert <转换前数值> <转换后单位>`

此命令可以使用 Pint 转换单位。单位符号大小写敏感。

:::info 例子

返回的结果示例：

```
>>> ~convert km mi
<<< 1 km = 0.6213711922373339696174341841 mi
```

:::
