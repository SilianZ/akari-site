# Translate

-   [开源]\(help/faq#Why some modules are not open sources)
-   Alias：`trans`

This module can translate text.

This module is based on [100 translations of API](https://fanyi-api.baidu.com/).

## `convert`

-   [公开](permissions)

Use：`~convert<pre-conversion values> <rear unit>`

This command can use the Pint conversion unit.Unit symbols are case-sensitive.

:::info Example

Result Example：

```
>>> ~convert km mi
<<<1 km = 0.6213711922373339696174341841 mi
```

:::
