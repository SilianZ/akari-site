# random

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Alias：`rand`, `rng`

This module can generate random numbers from within the range, or select a random element from the collection.

:::warning

This command is susceptible to abuse, please do not use too many times in a short period of time or trigger[滥用警告](abuse_warning).Abuse of command is a violation of[服务条款](policy), which not only affects normal communication, but can also cause robot accounts to be controlled.

:::

## `random number`

-   [公开](permissions)

Use：`~random number <small> <maximum>`

This command can generate an integer randomly from within the range.

## `random choice`

-   [公开](permissions)

Use：`~random choice <Element 1> <2> [<3> <Element 4>...]`

This command can select an element randomly from the collection.

## `random shuffle`

-   [公开](permissions)

Use：`~random shuffle <element 1> <element 2> [<3> <element 4>...]`

This command can rearrange all elements in the collection.

## `random password`

-   [公开](permissions)

Use：`~random password <length> [-u] [-l] [-n] [-s]`

Parameter：

-   `-u`：uses uppercase letters
-   `-l`：uses lowercase letters
-   `-n`：uses a number
-   `-s`：with special symbols

This command can generate a string of a certain length.

Strings are default as alphanumeric combinations. Supported special symbols are `!@#$%^&*-_+=?`.

## `random uuid`

-   [公开](permissions)

Use：`~random uid`

This command can generate a random UUID.
