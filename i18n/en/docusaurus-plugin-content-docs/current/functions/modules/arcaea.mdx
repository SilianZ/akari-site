# arcaea

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Alias：`a`, `arc`

This module can query information from the Arcea game.

:::info

According to Arcaea TOS 3.2.6, song splits will be disabled indefinitely, see[此处](https://arcaea.lowiro.com/en/terms_of_service).

:::

## `arcaea random`

-   [公开](permissions)

Use：`~arcaea random`

This module allows for a random Arcaea track

## `arcaea rank free`

-   [公开](permissions)

Use：`~arcaea rank free`

This module allows you to view the current free pack to play rows.

:::info Example

Result Example：

