# alias

-   [开源](/docs/help/faq#为什么有些模块没有开源)
-   强制开启

此模块可以自定义命令别名。

## `alias add`

-   [仅管理员](/docs/permissions)

用法：`~alias add <别名> <命令>`

此命令可以设置自定义机器人命令别名，被设置的命令同样需要命令前缀。

输入设置的命令别名即可激活对应命令。

## `alias remove`

-   [仅管理员](permissions)

用法：`~alias remove <别名>`

此命令可以移除自定义机器人命令别名。

## `alias reset`

-   [仅管理员](permissions)

用法：`~alias reset`

此命令可以重置自定义机器人命令别名。

## `alias list`

-   [仅管理员](permissions)

用法：`~alias list`

此命令可以列出自定义命令别名列表。
