# alias

-   [开源]\(/docs/help/faq#Why some modules are not open sources)
-   Force turn on

This module can customize the command alias.

## `alias add`

-   [仅管理员](/docs/permissions)

Use：`~alias add <Alias > <Command>`

This command can set a custom alias for robot command. Command prefix is required for command.

Enter the command alias set to activate the corresponding command.

## `alias remove`

-   [仅管理员](permissions)

Use：`~alias remove <alias>`

This command can remove custom robot alias.

## `alias reset`

-   [仅管理员](permissions)

Use：`~alias reset`

This command can reset custom robot alias.

## `alias list`

-   [仅管理员](permissions)

Use：`~alias list`

This command can list custom command aliases.
