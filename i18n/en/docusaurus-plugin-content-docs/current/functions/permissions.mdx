# Permissions

The following Groups of： are built-in

-   `superuser`：, a global robot administrator, is used by the developer only.
-   `admin`：Robot Admin, internal admin or private chat has this permission by default.
-   `user`：regular user.

The permissions logo in this document corresponds to the following：

| Identifier       | Competing Permissions             |
| ---------------- | --------------------------------- |
| Admin Only       | `superuser` 和 `admin`             |
| Superuser only   | `superuser`                       |
| Public           | `user`                            |
