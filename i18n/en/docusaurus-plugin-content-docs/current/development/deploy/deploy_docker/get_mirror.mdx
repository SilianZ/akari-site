# 拉取镜像

输入下面的指令拉取镜像。

:::info

目前小可的 Docker 镜像支持的架构仅为 arm64 和 amd64。

:::

```sh
docker pull bakabaka9/akari-bot:latest
```

:::info

该镜像的作者长期未更新，建议使用以下镜像

:::

```sh
docker pull silianz/akari-bot:dev-docker
```
