# Pull image

Enter the command below to pull the mirror.

:::info

The structures currently supported by the mini-docker mirrors are only arm64 and amd64.

:::

```sh
docker pull baka9/akari-bot:latest
```

:::info

The author of this image has not been updated for a long time. It is recommended to use the following image

:::

```sh
docker null silianz/akari-bot:dev-docker
```
