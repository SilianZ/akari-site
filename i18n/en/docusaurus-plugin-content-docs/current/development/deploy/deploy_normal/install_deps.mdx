# Install Dependencies

<Tabs groupId="operating-systems">
<TabItem value="win" label="Windows" default>
<Tabs>
<TabItem value="c1" label="使用 Poetry" default>

If you have installed Poetry, you can skip the following installation steps.

1.  Open Powershell, and execute the following instructions to install Poetry：

    ```powershell
    (Invoke-WebRequest -Uri "https://install.python-poetry.org" -UseBasicParsing).Content | my -
    ```

    :::info

    If you install Python using Microsoft Store or pyenv-windows, replace `python` with `python`.

    :::

    :::warning

    Before installing Poetry, **Make sure that** check that multiple Python Executable Paths (Python executable) exist in System Environment Variables and clear them in time. Otherwise, the installation of Poetry may cause a Python environment disrupting the load dependency.

    :::

2.  After the installation is complete, add the following directory to the PATH environment variable and make it easier to call：

    ```
    %APPDATA%\Python\Scripts
    ```

    :::info

    Use the search engine to find a way to change PATH.

    :::

    :::info

    After adding to PATH, you usually need to reboot the terminal, or even the entire computer.

    You can confirm whether the installation is valid via `poetry --version`.

    :::

3.  After installing Poetry, please execute the following directive：

    ```powershell
    post install
    ```

</TabItem>
<TabItem value="c2" label="使用 pip">

If you don't want to use Poetry, you can use pip to install dependency：

```powershell
pip install -r requirements.txt
```

</TabItem>
</Tabs>

:::info

Please go to[疑难解答](troubleshooting) if you report an error while installing dependency.

:::

</TabItem>
<TabItem value="linux" label="Linux">
<Tabs>
<TabItem value="c1" label="使用 Poetry" default>

If you have installed Poetry, you can skip the following installation steps.

1.  Open Terminal, and execute the following instructions to install Poetry：

    ```sh
    curl -sSL "https://install.python-poetry.org" | python3 -
    ```

    :::warning

    Before installing Poetry, **Make sure that** check that multiple Python Executable Paths (Python executable) exist in System Environment Variables and clear them in time. Otherwise, the installation of Poetry may cause a Python environment disrupting the load dependency.

    :::

2.  After the installation is complete, add the following directory to the PATH environment variable and make it easier to call：

    ```
    $HOME/.local/bin
    ```

    :::info

    Use the search engine to find a way to change PATH.

    :::

    :::info

    After adding to PATH, you usually need to reboot the terminal, or even the entire computer.

    You can confirm whether the installation is valid via `poetry --version`.

    :::

3.  After installing Poetry, please execute the following directive：

    ```sh
    post install
    ```

</TabItem>
<TabItem value="c2" label="使用 pip">

If you don't want to use Poetry, you can use pip to install dependency：

```sh
pip install -r requirements.txt
```

</TabItem>
</Tabs>

:::info

Please go to[疑难解答](troubleshooting) if you report an error while installing dependency.

:::
