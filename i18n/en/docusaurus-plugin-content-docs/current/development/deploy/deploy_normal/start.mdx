# Run robot

The small cocoa robot consists mainly of platform robots. In order for developers to better test modules, we have also provided a test console, and we will then show how to run the robots.

## Run Platform Robots

<Tabs groupId="operating-systems">
<TabItem value="win" label="Windows" default>

我们不推荐双击运行 `start.bat` 来启动程序。

建议在启动机器人之前，先打开终端（cmd 或 Powershell）再运行 `start.bat`。

1.  于 `start.bat` 所在目录，打开终端。
2.  在终端内输入 `./start.bat` （Powershell） 或 `start.bat` （cmd）来启动机器人。

</TabItem>
<TabItem value="linux" label="Linux">

1.  于终端内，设置 `start` 脚本的执行权限：

    ```sh
    chmod +x start
    ```

2.  启动脚本：`./start`。

</TabItem>
</Tabs>

## 运行测试控制台

测试控制台包括一个基础的运行环境，您可以在测试控制台内使用命令进行基础的机器人交互。

测试控制台仅支持回复文本消息和图片，其它消息元素将被忽略或转换为文本或图片来显示。

:::info

您可能需要使用 `poetry shell` 切换 poetry 的虚拟环境来调用先前安装的依赖。

:::

1.  于 `console.py` 所在目录，打开终端。
2.  在终端内输入 `python console.py` 来启动测试控制台。
