# Troubleshooting

The troubleshooter section below will solve the problems encountered by small parts when they are built.

Before troubleshooting, make sure you have read all the annotations in the document in detail.

Troubleshooter will be divided into different areas, if you have better trouble answering to submit PR.

## Install Dependencies

### Unable to install dependencies due to problems associated with `hnswlib`

This may only happen under Windows system.

You may not have installed Microsoft C++ generation tools.

1.  Download [Microsoft C++ Generating Tool](https://visualstudio.microsoft.com/en-hans/visual-cpp-build-tools/).

2.  Open the installer in the download folder, waiting for a minute to open a window called Visual Studio Installer.

3.  In the "Workload" block, select "Desktop Development using C++" and then click "Install" in the bottom right corner.

4.  Waiting for installation to close the window.

5.  Try reinstalling dependency.

### Dependency conflict when installing dependency using pip

Try adding the `--no-deps` parameter when installing dependencies.
