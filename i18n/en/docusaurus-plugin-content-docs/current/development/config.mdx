# Profile

The purpose of each parameter in the configuration file (`config.toml`) will be explained here.

Robot configuration file exists in `config` directory, generated after first start.The configuration file can be modified using the `~config` command and will take effect after reloading the module or restarting.

If you don't need some features, delete the corresponding key. Do not leave it blank.

## Properties

### Secret

| Properties                            | Use Modules        | Default value             | Description                                                                            |
| ------------------------------------- | ------------------ | ------------------------- | -------------------------------------------------------------------------------------- |
| db_path                               |                    | "mysql+pymysql:/"         | Database path used to store user data.                                                 |
| check_accessKeyId                     |                    |                           | The AccessKey ID required when calling the Aliyun Content Security Service API.        |
| check_accessKeySecret                 |                    |                           | The AccessKey Secretary required when calling the Aliyun Content Security Service API. |
| web_render                            |                    |                           | Webrender address (IP or domain name) and port.                                        |
| Proxy                                 |                    |                           | The address of the proxy service (IP or domain name) and the port.                     |
| dc_token                              |                    |                           | Discord Robot Token.                                                                   |
| kok_token                             |                    |                           | Kook robot token.                                                                      |
| tg_token                              |                    |                           | Telegram robot token.                                                                  |
| matrix_home server                    |                    |                           | Matrix server URL.                                                                     |
| matrix_user                           |                    |                           | Matrix Robot User ID (containing `@` and `:`).                                         |
| matrix_device_id                      |                    |                           | Matrix Robot Device ID.                                                                |
| matrix_device_name                    |                    |                           | Matrix robot device name.                                                              |
| Matrix_token                          |                    |                           | Matrix Robot Token.                                                                    |
| matrix_megolm_backup_password         |                    |                           | Matrix Robot's Megolm backup password phrase.                                          |
| openai_api_key                        | `ask` `summary`    |                           | OpenAI API key.                                                                        |
| curseforge_api_key                    | `mod_dl`           |                           | Curseforge API key.                                                                    |
| exchange_rate_api_key                 | `exchange_rate`    |                           | ExchangeRate-API key.                                                                  |
| wolfram_alpha_appid                   | `wolflramalpa`     |                           | The App ID of WolframAlpha API.                                                        |
