# 介绍

<div align="center">
<img width="300" src="https://s2.loli.net/2023/08/24/7HGPQ92Ro38yS6z.png" alt="logo"/>
## 小可 · AkariBot v4
*（图标由 [Rin 十一](https://mihuashi.com/profiles/75826?role=painter)绘制。）*
</div>

⚡ 小可是基于**多个框架开发**的**可扩展**、**多平台**机器人。

💅 通过开发者的努力，小可带来了多种**实用的模块**（wiki，ask 等），快来体验这些模块吧。

🎨 小可支持 QQ、Discord 等**多个平台**。

💥 您亦可[搭建](#自搭建)一个**属于自己**的小可！我们提供了 **Docker 镜像**和 **Poetry 包管理器**。

* * *

## 平台

小可目前支持以下平台：

-   QQ
-   Discord
-   Telegram
-   Kook
-   Matrix
