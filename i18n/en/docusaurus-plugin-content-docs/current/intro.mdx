# Information

## Introduction

**AkariBot** is an extensive, multi-platform robot developed based on multiple frameworks.图标由 [Rin 十一](https://mihuashi.com/profiles/75826?role=painter)绘制。

There is also ongoing development of AkariBot.

## Platforms

The following platforms are currently supported by: 

-   QQ
-   Discord
-   Telegram
-   Kook
-   Matrix
