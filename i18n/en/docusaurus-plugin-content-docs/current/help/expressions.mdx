# Math Expression

本页面为使用 Python 数学表达式的模块（如 [calc](/docs/calc)）提供帮助。

## Operator

### Numeric operator

| Operator | Meaning            | Example         |
| -------- | ------------------ | --------------- |
| `+`      | and                | `1 + 2 = 3`     |
| `-`      | Bad                | `3 - 1 = 2`     |
| `*`      | Product            | `2 * 3 = 6`     |
| `**`     | Power              | `2 ** 3 = 8`    |
| `/`      | Business           | `6 / 3 = 2`     |
| `//`     | Remove integration | `7 // 4 = 1`    |
| `%`      | Model              | `5 % 2 = 1`     |

### Comparator Operator

| Operator | Meaning                  | Example             |
| -------- | ------------------------ | ------------------- |
| `==`     | equals                   | `1 == 1 = True`     |
| `!=`     | Not equal to             | `1!= 1 = False`     |
| `<`      | Less than                | `1 < 2 = True`      |
| `>`      | greater than             | `2 > 1 = True`      |
| `<=`     | less than or equal to    | `1 <= 2 = True`     |
| `>=`     | Greater than or equal to | `2 >= 1 = True`     |

### by bit operator

| Operator | Meaning        | Example           | Binary                  |           |        |               |
| -------- | -------------- | ----------------- | ----------------------- | --------- | ------ | ------------- |
| `&`      | By bit with    | `10 & 12 = 8`     | `1010 & 1100 = 1000`    |           |        |               |
| \`       | \`             | By bit or         | \`10                    | 12 = 14\` | \`1010 | 1100 = 1110\` |
| `^`      | By bit isn't   | `10 ^ 12 = 6`     | `1010 ^ 1100 = 0110`    |           |        |               |
| `~`      | Reverse by bit | `~ 12 = - 13`     | `~ 1100 = 0011`         |           |        |               |
| `>>`     | Move Right     | `12 >> 2 = 3`     | `001100 >> 2 = 000011`  |           |        |               |
| `<<`     | Move left      | `12 << 2 = 48`    | `001100 << 2 = 110000`  |           |        |               |

### Logical Operator

| Operator | Meaning    | Example                     |
| -------- | ---------- | --------------------------- |
| `and`    | Logic with | `True and False = False`    |
| `or`     | Logic or   | `True or False = True`      |
| `not`    | Logic Not  | `not True = False`          |

### Decimal

[浮点数精度问题]\(https://en.wikipedia.org/wiki/floating point #accuracy) may lead to strange results, such as 3.30000000000003.If you have this problem, try using Decimal.

The float type is anti-human. It cannot even accurately represent 1.1 values.Decimal type “is designed to be based on floating point models that take into account human habits and therefore has the highest guiding principle that computers must provide arithmetic that is consistent with the arithmetic people learn in school”.

As to length, we do not dwell on the specific use of Decimal, at [official Python documents] \(https://docs.python.org/en-cn/3/library/decimal.html).Please note that neither of the mathematical functions above can be applied to Decimal, but the Decimal object has a built-in mathematical method.
