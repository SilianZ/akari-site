# Read Help

This page can help you read the document.

## Command format

Use the `~wiki` command as example：

```
~wiki [<interwiki>:]<page name>
```

`<页面名>` 代表**参数**。即使用 `<` 和 `>` 包含的参数，默认必填。

`[<interwiki>:]` 代表**选填参数**。即使用 `[` 和 `]` 包含的参数。Normally, the help document refers to default values if the optional parameter is not entered.

另外，带有 `(|)` 则代表**或**，即使用两种（及以上）选择的任意一种都可以响应命令。

Error Example

Do not imitate this command, it will not be answered correctly.

-   ~~\`~wiki [:]<rain >~~!

## Sample Command

命令示例有时会出现在页面中，以下是命令示例的格式，以 `~whoami` 命令作为示例：

:::info Example

Result Example：

```
>>>> ~whoami
<<<< your ID is：QQQ|xxxx
ID of this conversation is：QQQ|Group|xxx
```

其中，`>>>` 代表输入（即你发送给机器人的命令），`<<<` 代表命令的输出（即机器人发送给你的结果）。

:::

## 指导

指导通常会出现在文档和命令的开头。

### 是否开源

-   [开源](/docs/help/faq#为什么有些模块没有开源)：此模块可开源并按照协议使用。
-   [闭源](/docs/help/faq#为什么有些模块没有开源)：此模块的使用需要向模块原作者授权。

### 用户权限

-   [公开](/docs/permissions)：此命令可公开使用。
-   [仅管理员](/docs/permissions)：此命令仅机器人管理员或群管理员可使用。
-   [仅超级用户](/docs/permissions)：此命令仅超级用户可使用。

### 命令类型

-   [命令](/docs/help/reading#命令类型)：此命令在使用时会立刻做出回应。（一般不显示）
-   [订阅](/docs/help/reading#命令类型)：此命令在使用后可开关消息推送。
-   [设置](/docs/help/reading#命令类型)：此命令在使用后可更改模块设定。

### 其它

-   强制开启：此命令为基础命令，无法被手动禁用。
-   （任意时间）后撤回：此命令输出的信息会在若干时间后撤回。
-   冷却（任意时间）：使用此命令后在若干时间秒后才可再次使用。
-   不推荐：不建议用户使用此命令。
