If you have installed [Poetry](https://python-poetry.org/), you can skip the following installation steps.

1.  Open Powershell, and execute the commands below to install Poetryï¼š

    ```powershell
    (Invoke-WebRequest -Uri "https://install.python-poetry.org" -UseBasicParsing).Content | py -
    ```

    :::info

    If you install Python using Microsoft Store or pyenv-windows, replace `py` with `python`.

    :::

    :::caution

    Before installing Poetry, **Make sure** that check that multiple Python Executable Paths exist in System Environment Variables and clear them in time. Otherwise, the installation of Poetry may cause a Python environment to disrupt the load dependency.

    :::

2.  After the installation is complete, add the following directory to the PATH environment variable and make it easier to call:

    ```
    %APPDATA%\Python\Scripts
    ```

    :::info

    Please use the search engines to find ways to change PATH.

    :::

    :::info

    After adding to PATH, you usually need to reboot the terminal, or even restart the computer.

    You can confirm whether the installation is valid via `poetry --version`.

    :::

3.  After installing Poetry, please execute the following commands:

    ```powershell
    poetry install
    ```

If you have installed [Poetry](https://python-poetry.org/), you can skip the following installation steps.

1.  Open the terminal, and execute the following commands to install Poetry:

    ```sh
    curl -sSL "https://install.python-poetry.org" | python3 -
    ```

    :::caution

    Before installing Poetry, **Make sure** that check that multiple Python Executable Paths exist in System Environment Variables and clear them in time. Otherwise, the installation of Poetry may cause a Python environment to disrupt the load dependency.

    :::

2.  After the installation is complete, add the following directory to the PATH environment variable and make it easier to call:

    ```
    $HOME/.local/bin
    ```

    :::info

    Please use the search engines to find ways to change PATH.

    :::

    :::info

    After adding to PATH, you usually need to reboot the terminal, or even restart the computer.

    You can confirm whether the installation is valid via `poetry --version`.

    :::

3.  After installing Poetry, please execute the following commands:

    ```sh
    poetry install
    ```
