# Preparation

1.  A server or host that can run Python (computer, Raspberry Pi, Termux mobile phone, etc...)
2.  The host is installed and can run [Python 3 environment](https://www.python.org/). Version greater than 3.8 can be deployed.
3.  Corresponding to the necessary content (environment, token, etc.) required for the platform you need to run.
