# Download Source Code

```mdx-code-block
<Tabs groupId='operating-systems'>
<TabItem value="win" label="Windows" default>
<Tabs>
<TabItem value="c1" label="Clone with Git" default>
```

1.  Install [Git](https://git-scm.com/).

2.  Please right-click and open Git Bash in a folder to place an AkariBot and enter the following command: 

    ```sh
    git clone https://github.com/Teahouse-studios/akari-bot.git
    ```

    Cloning at GitHub is slower. In exceptional cases, you can clone using a mirror station:

    ```sh
    git clone https://gitclone.com/github.com/Teahouse-studios/akari-bot.git
    ```

    :::info

    The mirror station will update the repository every night, so please use the original GitHub address if you want to get the latest version of the code before the mirror station is updated.

    :::

3.  Press `Shift` + `Right-click` and open the terminal.

```mdx-code-block
</TabItem>
```

1.  下载 [master 分支的最新代码](https://github.com/Teahouse-Studios/akari-bot/archive/refs/heads/master.zip)。

    :::info

    由于无法保证 Release 版本的稳定性，我们不再提供 Release 版本的下载。

    :::

    :::info

    不建议直接下载代码部署，因为这样您将无法使用 Git 或是使用机器人内置的命令更新代码。

    :::

2.  解压源代码，按住 Shift + 右键，选择 在此处打开 Powershell 窗口 或 在此处打开命令窗口 来打开终端，进入文件夹。

    :::info
